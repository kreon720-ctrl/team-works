## 스토리보드 컷 구성

### Act 1. 멀티태스킹의 카오스 (0~6초)

#### Cut 1 — 00:00~00:01.5 / 오프닝 와이드 숏

| 항목 | 내용 |
|------|------|
| 샷 | Wide Shot, 정면 앵글, 살짝 하이앵글(15°) |
| 비주얼 | 어두운 오피스. 30대 직장인(남/여 2인조)이 모니터 2대 앞에 앉아 있고 책상 위에는 노트북·태블릿·스마트폰이 중첩 배치. 화면마다 슬랙·노션·엑셀·구글캘린더 창이 무질서하게 띄워져 있다. 화면 빛이 얼굴에 파랗게 번진다. |
| 모션 | 카메라가 책상 쪽으로 천천히 돌리(dolly-in), 동시에 화면 속 창들이 1초에 3~4번 깜빡이며 알림 팝업이 연속으로 뜸 |
| 자막 | 없음 |
| 사운드 | 로파이 비트 + 타이핑 SFX + 알림 "띠링" 중첩(무음 버전은 파형 그래픽으로 시각화) |

#### Cut 2 — 00:01.5~00:03 / 화면 클로즈업 몽타주

| 항목 | 내용 |
|------|------|
| 샷 | ECU(Extreme Close-Up) 4분할. 각 분할에 슬랙·노션·엑셀·캘린더 화면 |
| 비주얼 | 각 화면에서 쏟아지는 알림, 읽지 않은 메시지 카운터가 `12 → 47 → 83 → 126`으로 급증. 엑셀 셀에는 `프로젝트_v7_최종_진짜최종.xlsx`라는 파일명이 잠깐 스쳐 지나감 |
| 모션 | 4분할이 순차적으로 화면을 잠식해 들어오며 프레임을 가득 채움. 화면이 미세하게 떨림(shake amplitude 2px) |
| 자막 | **"슬랙.  노션.  엑셀.  캘린더..."** (한 단어씩 0.3초 간격 타이핑) |
| 사운드 | 타이핑·마우스 클릭·알림음이 점점 겹쳐 불협화음 형성 |

#### Cut 3 — 00:03~00:04.5 / 인물 리액션 숏

| 항목 | 내용 |
|------|------|
| 샷 | Medium Close-Up, 아이레벨 |
| 비주얼 | 주인공이 한숨을 쉬며 관자놀이를 누르는 모습. 눈 아래 다크서클이 살짝 보이고, 배경에는 모니터 빛이 블러 처리되어 정신없이 번쩍인다 |
| 모션 | 살짝 핸드헬드 느낌(1Hz 미세 떨림)으로 피로감 강조 |
| 자막 |  **"업무에 지친 당신"** |
| 사운드 | 불협화음이 정점에 도달, 심박 같은 베이스 SFX 추가 |

#### Cut 4 — 00:04.5~00:06 / 탭 폭발 샷

| 항목 | 내용 |
|------|------|
| 샷 | Over-the-Shoulder, 모니터 중심 |
| 비주얼 | 브라우저 탭이 30개 이상 나열되어 있고, 그 아래 노션·슬랙·엑셀 앱 아이콘이 독(dock)에 가득 차 있다. 탭 하나가 `X`자 모양으로 붉게 깜빡이며 "응답 없음" 경고 |
| 모션 | 화면 전체가 진동(zoom 1.0 → 1.02 반복), 붉은 경고 아이콘이 프레임 중앙을 차지. 6초 시점에 화면이 순간 블랙아웃 |
| 자막 | **"쉴틈 없이 바쁜 하루"** |
| 사운드 | 불협화음이 갑작스러운 정적으로 전환(컷 오프) |

### Act 2. 솔루션 등장 — TEAM WORKS (7~11초)

#### Cut 5 — 00:06~00:07 / 블랙아웃 → 라이트 온

| 항목 | 내용 |
|------|------|
| 샷 | 풀 프레임 블랙 → 페이드 인 |

| 사운드 | 로파이 비트가 완전히 사라지고 잔잔한 피아노 솔로가 시작. SKT 오마주 포인트 |

#### Cut 6 — 00:07~00:09 / UI 전환 모션그래픽

| 항목 | 내용 |
|------|------|
| 샷 | 화면 가득 채우는 UI 디스플레이 숏 |
| 비주얼 | Cut 4의 어지러운 창 15개가 각기 다른 방향에서 중앙으로 빨려들어와 TEAM WORKS 화면으로 대체됨 |
| 모션 | 흩어진 창들이 2초에 걸쳐 중앙으로 수렴(magnetic pull), 각 요소가 순차적으로 스냅(snap) 정렬 |
| 자막 | **"프로젝트 워크벤치. 스마트 캘린더. 팀채팅."** |
| 사운드 | 각 창이 합쳐질 때마다 부드러운 "톡" SFX, 최종 수렴 시 따뜻한 "완료" 벨 |

| 모션 | 슬로모션(0.8x), 카메라가 아주 천천히 푸시 인. 주인공 뒤 창문으로 아침 햇살이 들어옴 |
