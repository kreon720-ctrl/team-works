## Q1. 개인 일정도 관리할 수 있나요?

### A. 네 — **비공개 팀** 을 하나 만들어 혼자 쓰시면 됩니다

TEAM WORKS 는 "팀" 단위로 모든 일정·메모가 묶이지만, **혼자만 쓰는 비공개 팀** 도 자유롭게 만들 수 있어요.

#### 만드는 법

```
[좌측 햄버거 메뉴 ≡] → [새 팀 만들기]
   ↓
  팀 이름:  "내 일정"  /  "Personal"  /  "혼자만의 다이어리"  등
  공개 여부: ○ 공개   ●︎ 비공개   ← 비공개 선택!
   ↓
  [만들기]
   ↓
  자동으로 팀장이 됨 — 다른 사람이 검색·가입 신청 못 함
```

#### 비공개 팀에서 쓸 수 있는 기능

| 기능 | 가능 여부 |
|------|----------|
| 일정 등록·조회·수정·삭제 | ✓ 동일 |
| 포스트잇 (메모) | ✓ 동일 |
| AI 찰떡이 자연어 명령 | ✓ 동일 |
| 음성 입력 | ✓ 동일 |
| 자료실 (첨부파일 보관) | ✓ 동일 |
| 팀채팅 | ✓ (혼잣말 메모용) |

> 💡 **Tip** — 회사용 공개 팀 + 개인용 비공개 팀을 동시에 운영하면 일·생활 일정을 한 화면에서 분리해 볼 수 있어요. 팀별로 캘린더가 따로 보관됩니다.

## Q2. 주소창에 `teamworks.my` 치면 구글 검색 페이지로 가요

### A. 앞에 **`https://`** 를 붙여주세요 — `https://teamworks.my`

`.my` 도메인은 `.com` 처럼 브라우저가 익숙한 형태가 아니라, 주소가 아닌 **검색어** 로 인식되는 경우가 있어요. 특히 크롬·엣지의 omnibox(주소+검색 통합) 때문입니다.

#### 해결 방법

```
✗ 안 됨:                     ✓ 됨:
┌──────────────────────┐    ┌──────────────────────────┐
│  teamworks.my        │    │  https://teamworks.my    │
└──────────────────────┘    └──────────────────────────┘
   ↓ Enter                    ↓ Enter
   구글 검색결과로 이동       정상 접속!
```

#### 한 번만 접속하면 다음부터는 편해요

- 첫 접속 후 **북마크(즐겨찾기)** 에 저장하면 클릭 한 번으로 끝
- 또는 브라우저 시작 페이지로 설정
- 모바일에서는 **홈 화면에 추가** (앱처럼 아이콘으로 사용)

> 💡 **Tip** — 한 번 `https://` 붙여서 정상 접속하면, 다음부터는 주소창에 `teamw` 만 쳐도 자동완성으로 뜹니다.

## Q3. 음성 입력이 중간에 끊겨요

### A. **마이크를 다시 한 번 누르고 이어서 말하면** 자동으로 붙어요

휴대폰 음성 입력은 짧은 침묵에도 자동으로 멈추는 경우가 있는데요, 끊겨도 괜찮아요. 마이크 버튼을 다시 누르고 이어서 말하면 **기존 텍스트 뒤에 자동으로 이어 붙습니다**.

#### 동작 흐름

```
1)  🎤 클릭  →  "오늘 오후 3시"  →  (자동 멈춤)
                  ↓
        입력창: "오늘 오후 3시"

2)  🎤 다시 클릭  →  "주간회의 등록해줘"  →  (자동 멈춤)
                  ↓
        입력창: "오늘 오후 3시 주간회의 등록해줘"   ← 자동 이어붙음

3)  검토 → [전송] 클릭
```

#### 처음부터 다시 말하고 싶을 땐?

입력창 우측 안쪽의 작은 **✕ (클리어 버튼)** 을 누르면 비워집니다. 음성으로 잘못 인식된 경우 빠르게 다시 시작 가능.

```
┌──────────────────────────────────────────┬─────┐
│ 텍스트 ...                            ✕  │ 🎤  │  ← 텍스트 있을 때만 ✕ 표시
└──────────────────────────────────────────┴─────┘
```

#### 키보드 수정도 함께 가능

음성으로 한 큰 틀을 입력하고, 잘못 인식된 부분만 키보드로 다듬으면 가장 빠릅니다.

> 💡 **Tip** — 한 번에 길게 말하기 부담스러우면 짧게 끊어서 여러 번 누르세요. 결과는 한 줄로 자연스럽게 합쳐집니다.

## Q4. 어떤 인터넷 브라우저를 써야 하나요?

### A. 대부분 지원합니다 — **디바이스별 권장 브라우저** 만 알아두면 됩니다

| 디바이스 | 권장 브라우저 | 비고 |
|----------|--------------|------|
| **안드로이드 폰/태블릿** | **Chrome** ⭐ | 음성 인식·푸시 알림 최적화 |
| **아이폰 / 아이패드** | **Safari** ⭐ | iOS 네이티브 음성 엔진과 가장 잘 맞음 |
| **윈도우 PC** | Chrome / Edge | 둘 다 동등하게 동작 |
| **맥** | Safari / Chrome | 둘 다 동등하게 동작 |
| **리눅스** | Chrome / Firefox | 동작 OK |

#### 왜 디바이스마다 권장이 다른가요?

특히 **음성 입력** 은 OS 가 제공하는 음성 엔진(Apple/Google)을 브라우저가 위임 호출하는 구조라서, 디바이스 native 브라우저가 가장 정확하고 빠릅니다.

```
[안드로이드 + 크롬]   →  Google 음성 엔진 (최상)
[아이폰 + 사파리]      →  Apple 음성 엔진 (최상)
[안드로이드 + 사파리?] →  사파리는 iOS 전용
[아이폰 + 크롬]        →  사파리 엔진 위임 (Safari 와 동일)
[삼성 인터넷]          →  자체 엔진이 관여해 정확도 떨어짐 → Whisper 자동 분기
```

> 💡 **Tip** — 평소 다른 브라우저(Firefox, Brave 등)를 쓰셔도 텍스트 입력·일정 관리 같은 기본 기능은 모두 동작합니다. 음성 입력 정확도 차이만 있을 뿐.

## Q5. AI 찰떡이가 일정 명령을 잘못 알아들어요

### A. **[시점] + [목적어]** 를 명확히 — 한 줄에 한 가지 의도만

찰떡이는 자연어를 잘 이해하지만, 너무 짧거나 모호한 표현은 **조회인지 등록인지 헷갈려** 잘못 분류할 수 있습니다.

#### ✗ 잘못된 예시 — 의도가 모호

| 입력 | 문제 |
|------|------|
| *"일정 등록 조회"* | 등록인지 조회인지 동시 의도? |
| *"회의"* | 시점·동사 모두 없음 → 분류 실패 |
| *"내일"* | 동사 없음 → 무엇을 하라는 건지 불명 |

#### ✓ 잘 인식되는 패턴 — **[시점] + [목적어] + [동사]**

```
┌─────────┬──────────┬──────────┐
│  시점   │  목적어   │   동사    │
├─────────┼──────────┼──────────┤
│ 오늘     │ 점심 약속 │ 조회해줘  │
│ 내일 오후│ 회의 일정 │ 삭제해줘  │
│ 5월 22일 │ 디자인 리뷰 │ 등록해줘  │
│ 다음 주  │ 모든 회의 │ 보여줘    │
└─────────┴──────────┴──────────┘
```

#### 의도별 동사 키워드

| 의도 | 동사 키워드 | 예시 |
|------|------------|------|
| **조회** | 알려줘 / 보여줘 / 조회 / 확인 | *"오늘 일정 알려줘"* |
| **등록** | 등록 / 추가 / 잡아줘 / 생성 | *"내일 3시 회의 등록해줘"* |
| **삭제** | 삭제 / 지워 / 제거 | *"5월 14일 회의 삭제해줘"* |
| **수정** | 수정 / 변경 / 바꿔 / 옮겨 | *"내일 회의 수정해줘"* |

#### 답변이 모호하면 찰떡이가 다시 물어요

찰떡이가 정보가 부족하면 *"몇 시에 잡을까요?"*, *"어떤 일정을 삭제할까요?"* 같이 후속 질문을 던집니다. 이때는 짧게 답해도 됩니다.

```
사용자: "내일 회의 삭제"
찰떡이: "어떤 회의를 삭제할까요?"
   • 5. 15. 14:00 ~ 15:00  디자인 리뷰 회의
   • 5. 15. 17:00         주간 회의
사용자: "디자인 리뷰"
찰떡이: "다음 일정을 삭제할까요? • 5. 15. 14:00 ~ 15:00 디자인 리뷰 회의"  → ✓
```

> 💡 **Tip** — 등록·삭제·수정은 **미리보기 카드 + ✓** 단계가 한 번 더 있어서, 잘못 인식돼도 ✗ 누르면 안 됩니다. 안전하게 시도해 보세요.

# 추가로 자주 묻는 질문 10개

신규 사용자가 두 번째로 자주 만나는 질문들입니다.

## A1. 팀은 어떻게 만들고, 어떻게 가입해요?

### 직접 만드시면 됩니다 — 만든 사람이 자동으로 팀장이 돼요

#### 새 팀 만들기

1. 화면 좌측 상단 **햄버거 메뉴(≡)** 탭 → **새 팀 만들기**
2. 팀 이름·간단한 설명을 적어요
3. **공개 / 비공개** 둘 중 하나 선택
   - **공개 팀**: 다른 사람들이 검색해서 가입 신청을 보낼 수 있어요
   - **비공개 팀**: 내가 초대한 사람만 들어옵니다 ([Q1 참고](#q1-개인-일정도-관리할-수-있나요))
4. 만들면 끝! 자동으로 **팀장**이 됩니다

#### 다른 팀에 가입하기

- 좌측 메뉴에서 **팀 찾기** → 관심 있는 공개 팀 선택 → **가입 신청**
- 팀장이 승인하면 그 팀의 캘린더·채팅·자료에 바로 접근 가능

#### 팀장이 되면 뭘 할 수 있나요?

- 가입 신청을 **승인 / 거절** (좌측 메뉴 **나의 할 일** 에 모여 있어요)
- 팀명·설명·공개 여부 **언제든 수정**
- 팀에 안 맞는 팀원이 있으면 **팀에서 빼기**
- 한 사람이 **여러 팀의 팀장**이 될 수도 있고, 어떤 팀에서는 그냥 팀원일 수도 있어요

> 💡 회사 팀 + 사이드 프로젝트 + 동호회까지 한 계정으로 동시 운영 가능. 팀별로 캘린더·채팅·자료가 따로 보관돼요.

## A2. '일정'과 '포스트잇' 차이가 뭐예요?

### **시간 약속**이면 일정, **가벼운 메모**면 포스트잇

| 구분 | 시간 | 용도 | 등록 방법 |
|------|------|------|----------|
| **일정** | 시·분 단위로 명확 (또는 시작만) | 회의·마감·약속 | + 버튼 또는 AI 찰떡이 |
| **포스트잇** | 날짜만 | 잡일·메모·체크리스트 | 색상 선택 → 날짜 탭 |

#### 일정 (Schedule)
- 예: *"5월 14일 17:00 ~ 18:00 디자인 리뷰"*
- **5가지 색상**으로 구분 (회의는 indigo, 마감은 rose 같은 팀 규약)
- 캘린더 우측 상단 **노란색 + 버튼** 으로 등록

#### 포스트잇 (Post-It)
- 예: *"퇴근시 케익 사갈 것"*, *"내일 거래처 미팅 자료 챙기기"*
- 캘린더 위쪽 **포스트잇 색상 팔레트**(🟣🟡🔵🟢🔴) → 색 선택 → 날짜 탭
- 모바일에선 포스트잇 탭 시 **확대 팝업** 으로 크게 보고·편집

> 💡 *"내일 점심 12시 김부장님과 식사"* → **일정**. *"내일 점심 사가기"* → **포스트잇**.

## A3. 다른 사람 일정·포스트잇을 제가 지울 수도 있나요?

### 아니요 — **만든 사람 본인만** 수정·삭제

| 행동 | 누가 할 수 있나 |
|------|----------------|
| 일정·포스트잇 **올리기** | 팀원이라면 누구나 |
| 일정·포스트잇 **보기** | 팀원이라면 누구나 |
| **수정·삭제** | 올린 사람 본인만 |

화면에서 직접 클릭하든, AI 찰떡이한테 부탁하든 **똑같이** 적용됩니다.

#### 팀장은 다른가요?

팀장이라고 다른 사람의 일정을 마음대로 지울 권한은 **없습니다**. 팀장의 권한은 팀 운영(가입 승인·팀원 관리·팀 정보 수정) 에 한정.

> 💡 누가 만든 일정인지 확인하고 싶으면 일정을 탭해 보세요. 작성자 이름이 표시됩니다.

## A4. AI 찰떡이한테 뭘 시킬 수 있어요?

### 일정 관리 + 사용법 질문 + 일반 검색

찰떡이는 우측 상단 **✨ AI 찰떡이** 탭에서 만날 수 있어요.

| 의도 | 예시 |
|------|------|
| **일정 조회** | *"오늘 일정 알려줘"*, *"이번 주 회의 몇 건이야?"* |
| **일정 등록** | *"내일 오후 3시 주간회의 등록해줘"*, *"22일 11시 반 점심"* |
| **일정 삭제** | *"내일 회의 삭제해줘"* |
| **일정 수정** | *"내일 회의 수정해줘"* → 찰떡이가 단계별로 물어봅니다 |
| **사용법 질문** | *"포스트잇 색깔은 몇 종류야?"*, *"비공개 팀이 뭐야?"* |
| **일반 질문/웹 검색** | *"오늘 뉴스 검색해줘"*, *"돈까스 만드는 법"* |

#### 안전 장치

등록·삭제·수정은 항상 **미리보기 카드 + ✓ 승인 / ✕ 취소** 가 한 번 뜹니다. ✓ 눌러야 진짜 반영.

#### 못 하는 것

- 프로젝트 만들기·채팅 메시지 보내기는 **화면에서 직접**
- 일정 색깔·설명 변경, 일정 취소 같은 일부 수정은 화면에서 직접

> 💡 표현이 헷갈리면 [Q5](#q5-ai-찰떡이가-일정-명령을-잘못-알아들어요) 참고.

## A5. 마이크 버튼은 어떻게 써요?

### 한 번 탭 → 말하기 → 자동 멈춤 → [전송]

- **모바일 AI 찰떡이** 와 **모바일 팀 채팅** 입력창 옆 🎤 마이크 버튼
- PC 에서는 키보드가 빠르니 마이크 버튼은 모바일에서만 노출

#### 사용 흐름

1. 마이크 탭 → **빨간 펄스** (듣는 중)
2. *"오늘 3시 회의 등록"* 발화
3. 잠깐 침묵 → 자동 종료 → 입력창에 텍스트 채워짐
4. 검토 후 **[전송]**

#### 끊겼을 때

마이크를 다시 누르고 이어 말하면 자동으로 붙어요 ([Q3 참고](#q3-음성-입력이-중간에-끊겨요)).

#### 인식이 잘 안 될 때

- 마이크 권한이 차단됐는지 확인 (브라우저 설정)
- 조용한 곳에서 다시 시도
- 키보드로 부분 수정 가능

> 💡 디바이스별 권장 브라우저는 [Q4](#q4-어떤-인터넷-브라우저를-써야-하나요) 참고.

## A6. 다크 모드 / 라이트 모드는 어떻게 바꾸나요?

### 우측 상단 **토글 스위치** 한 번 클릭

```
┌──────────────────────────────────────────────┐
│   기획팀장   [3]   ⏻   ⚪⚫    ⋮             │  ← 헤더 우측의 토글
└──────────────────────────────────────────────┘
                       ↑
              밝은 모드 ↔ 어두운 모드
```

- **라이트 모드**: 밝은 배경 + 진한 글자 (낮·실내용)
- **다크 모드**: 어두운 배경 + 부드러운 글자 (밤·OLED 절전용)
- 선택은 **브라우저별로 기억** 되며, 새로고침 후에도 유지됩니다

> 💡 일정 색상(indigo·blue·emerald·amber·rose) 도 두 모드 모두에서 잘 보이게 자동 보정돼 있어요.

## A7. 종료 시각 없이도 일정 등록할 수 있나요?

### 네 — **시작만** 정해두고 종료는 비워둬도 됩니다

종료 시각이 정해지지 않은 일정 (예: *"19시에 만나기로 약속"*) 도 그대로 등록 가능합니다.

#### 화면에서 등록할 때

```
일정 등록 모달:
  제목 *        : 회식
  시작 일시 *   : 2026-05-22 19:00
  종료 일시 (선택): [비워둠]   ← 비워두면 시작시각만 정해진 일정으로 등록
  ↓
  [등록]
```

#### AI 찰떡이로 등록할 때

```
사용자: "5월 22일 19시 회식 등록해줘"
찰떡이: 다음 내용으로 일정 등록할까요?
   • 제목: 회식
   • 시작: 2026. 5. 22. PM 7:00:00
   ✓ / ✗                 ← "종료" 줄이 안 나옴 (선택 입력)
```

#### 표시는 어떻게 보이나요?

- **MonthView 칩**: `19:00 회식` (~ 기호 없음)
- **WeekView/DayView**: 시작시각 위치에 텍스트 분량만큼 작은 카드
- **AI 답변**: `5. 22. 19:00 회식` (~ 없음)

> 💡 종료시각을 명시한 일정과 자연스럽게 한 화면에 섞여 표시됩니다.

## A8. 가입 신청 승인은 어디서 하나요?

### 좌측 메뉴 **나의 할 일** 에서 한 번에

여러 팀의 팀장이라도 모든 가입 신청이 한 곳에 모여요.

```
[좌측 햄버거 ≡]
   ↓
   [나의 할 일]
   ├─ 가입 신청 (3건)         ← 빨간 배지 숫자
   │   • 김XX → 기획팀
   │   • 이XX → 사이드프로젝트
   │   • 박XX → 기획팀
   │      [승인] [거절]
   └─ ...
```

- 헤더 상단에도 **빨간 배지 숫자** (uid 의 "3") 로 새 신청 알려줌
- 한 번에 여러 신청 승인 가능

> 💡 비공개 팀에는 가입 신청 자체가 안 와요 (검색 안 됨).

## A9. 한 일정이 여러 날에 걸쳐도 등록 가능한가요?

### 네 — 시작·종료 일자를 다른 날로 설정하면 됩니다

#### 예: 5월 15일 ~ 5월 19일 출장

```
일정 등록 모달:
  제목         : 부산 출장
  시작 일시 *  : 2026-05-15 09:00
  종료 일시    : 2026-05-19 18:00   ← 시작과 다른 날
```

#### 표시 방식

- **MonthView**: 5/15 ~ 5/19 까지 색상 띠로 가로지르며 표시 (멀티데이 밴드)
- **WeekView**: 상단 "종일/다일" 영역에 한 줄로 표시
- **AI 답변**: `5. 15. 09:00 ~ 5. 19. 18:00 부산 출장` (날짜 다르면 양쪽 모두 month/day 표시)

> 💡 AI 찰떡이로도 *"5월 15일부터 19일까지 부산 출장 등록해줘"* 처럼 자연어로 가능합니다.

## A10. 모바일에서 캘린더·팀채팅·AI 찰떡이는 어떻게 전환하나요?

### 화면 상단 **3개 탭** 으로 전환

```
┌────────────┬────────────┬────────────┐
│  📅 캘린더  │ 💬 팀채팅  │ ✨ AI 찰떡이│  ← 탭 한 번 클릭으로 전환
└────────────┴────────────┴────────────┘
```

- **캘린더**: 월·주·일 뷰 + 포스트잇
- **팀채팅**: 일자별 / 프로젝트별 채팅 + 자료실 + 공지사항
- **AI 찰떡이**: 자연어 일정 명령 + 사용법 질문 + 웹 검색

#### 좌우 스와이프로도 전환 가능

탭을 누르는 대신 화면을 **좌우로 쓱** 밀어도 같은 탭들이 전환됩니다 (모바일 한 손 조작 친화).

#### AI 찰떡이 안에서 캘린더 보기

AI 찰떡이 입력창 우측 끝 **📅 캘린더 아이콘** 을 누르면 화면 위쪽에 캘린더가 함께 떠서 답변 결과를 즉시 시각적으로 확인할 수 있습니다.

> 💡 PC 에서는 캘린더 + 팀채팅이 좌우로 동시에 표시되고, AI 찰떡이는 우측 상단 탭으로 전환 가능합니다.
