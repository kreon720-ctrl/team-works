# TEAM WORKS

> **카톡, 슬랙, 노션, 엑셀… 이제 꺼두셔도 좋습니다...**
> **AI와 함께하는 올인원 스마트 팀 워크스페이스, TEAM WORKS**

팀웍스는 **팀 빌딩 → 일정 관리 → 프로젝트 관리 → 자료 공유 → 보고**까지 하나의 화면에서 업무 플로우를 모두 지원하는 토털 솔루션입니다. 거기에 일정을 자연어로 **조회·등록·삭제·수정**까지 대신 처리해주고, 사용법이 헷갈리면 곧장 답해주는 AI 버틀러 **찰떡이**까지. "그래서 그 회의 몇 시였지?" 단톡방 스크롤 무한 내리던 날들이여, 이제 안녕.


- **팀 빌딩** — 내가 직접 팀을 만들고, 가입 신청도 내가 골라서 받기
- **스마트 캘린더** — 월·주·일 뷰 + 컬러 포스트잇으로 한눈에
- **프로젝트 워크벤치** — 3단계 간트차트, 진행률·지연 자동 표시
- **팀 채팅** — 일자별·프로젝트별로 깔끔히 분리. 회의록·자료까지 한 곳에
- **AI 버틀러 찰떡이** — 자연어로 일정 조회·등록·삭제·수정, 사용법·웹 검색까지

## Why TEAM WORKS

### 1. 여러 앱 왔다 갔다, 이제 끝
슬랙에서 일정 얘기하고, 구글 캘린더에 옮기고, 노션에 정리하고, 엑셀로 간트차트 그리고, 단톡방에서 자료 받고… 하루에 몇 번이나 반복하셨어요? 팀웍스 한 탭만 켜두면 일정·프로젝트·채팅·자료까지 모든 흐름이 한 화면 안에서 굴러갑니다.

### 2. 팀 빌딩부터 보고까지, 원스톱
새 팀 꾸리기 → 팀원 모으기 → 일정 짜기 → 프로젝트 진행 → 보고까지. 팀이 시작하고 끝낼 때까지 필요한 모든 단계가 한 앱 안에 있어요.

### 3. 복잡한 업무 흐름, 한 장으로 정리
2단계 프로젝트 뷰로 큰 그림과 디테일을 동시에 봅니다. 진행률은 %로, 늦어진 일은 색이 바뀌어 자동으로 "여기 빨간불"이라고 알려줘요. `프로젝트_일정_v7_최종_진짜최종.xlsx` 만들던 야근, 작별 인사 드릴 시간.

### 4. 팀장도 팀원도 숨통이 트입니다
팀장은 업무보고 조회 권한을 직접 정할 수 있어 민감한 보고가 엉뚱한 사람한테 흘러갈 일이 없어요. 여러 팀을 동시에 이끌어도 **나의 할 일** 한 화면에서 가입 신청을 모아 처리할 수 있어 이리저리 옮겨다닐 필요가 없습니다.

### 5. 지난주 결정사항? 1초 컷
회의·메모·대화가 날짜별로 자동 정리됩니다. 며칠 전 결정사항을 다시 꺼내거나 주간 회고할 때 단톡방 스크롤 무한히 내릴 필요 없이 날짜만 찍으면 끝. 업무처리에 필요한 자료공유와 대화를 한화면에서

### 6. AI 버틀러가 옆에 있습니다 — 사용법, 웹검색, 일정 4기능 모두
"내일 오후 3시 주간회의 등록해줘" / "5월 14일 회의 삭제해줘" / "내일 회의 수정해줘" — 한 줄이면 AI 버틀러 **찰떡이**가 일정을 잡고·지우고·바꿔드립니다. **등록·삭제·수정 모두 미리보기 카드 → ✓ 클릭** 후에만 실제로 반영되니 실수도 없어요. 사용법이 헷갈릴 땐 그냥 자연어로 물어보세요. 개떡같은 질문에도 찰떡같이 답해줍니다.

## What's Inside

### 1. 팀 빌딩 — "내 팀, 내가 만들고 운영해요"

새 프로젝트 시작할 때마다 IT팀에 "채널 만들어 주세요" 부탁하거나, 단톡방 새로 파서 사람 초대하느라 진땀 빼던 경험 있으시죠. 팀웍스는 **내가 직접 팀을 만들고, 만든 사람이 자동으로 팀장**이 됩니다.

- **자유로운 팀 구성** — 팀명·설명을 정하고, 공개로 둘지 비공개로 둘지도 직접 선택. 공개 팀이면 다른 사람들이 직접 가입 신청을 해와요.
- **승인은 한 곳에서** — 여러 팀을 이끌고 있어도 **나의 할 일** 화면에 가입 신청이 전부 모여, 한 번에 받을 사람만 골라 승인하면 끝.
- **팀 단위 격리** — 한 팀의 일정·채팅·자료실은 그 팀 안에서만 보입니다. 다른 팀 일은 절대 새지 않아요.
- **운영 권한도 팀장 손에** — 팀명·설명·공개 여부는 언제든 수정. 정중히 작별할 팀원이 있다면 명단에서 빼는 것도 가능합니다.
- **한 사람, 여러 팀** — 한 계정으로 회사 팀 + 사이드 프로젝트 팀 + 동호회 팀까지 동시에. 어떤 팀에서는 팀장, 다른 팀에서는 팀원도 자유롭게.

### 2. 스마트 캘린더 — "팀 일정, 더 이상 헷갈리지 않아요"

"다음 주 수요일 회의 몇 시였지?" 단톡방 스크롤 끝까지 내려본 적, 다들 한 번쯤 있죠.

- **월·주·일 뷰** — 이번 달 전체 그림은 월 뷰로, 이번 주 시간표는 주 뷰로, 오늘 빡빡한 일정은 일 뷰로. 보고 싶은 대로 골라 보세요.
- **누구나 등록, 본인만 수정·삭제** — 팀원이라면 누구나 일정을 추가할 수 있고, 내가 올린 일정만 내가 수정·삭제할 수 있어요. 누가 마음대로 회의 시간을 지워버릴 걱정 없음. 화면이든 AI 찰떡이든 동일하게 **생성자 본인만** 변경 가능합니다.
- **5가지 색상으로 구분** — indigo, blue, emerald, amber, rose. "회의는 indigo, 마감은 rose" 같은 팀 규약을 정해두면 캘린더만 봐도 뭐가 뭔지 바로 보입니다.
- **컬러 포스트잇** — "내일 거래처 미팅 전에 자료 챙기기!" 같은 가벼운 메모, 색깔 골라서 날짜에 척 붙여둘 수 있어요. 시간 정해진 일정은 일정으로, 그 날 챙길 잡일은 포스트잇으로.
- **놓칠 수 없는 검증** — 종료 시각이 시작 시각보다 앞이면 자동으로 막아줍니다. 실수로 회의 끝나는 시간을 시작보다 빠르게 잡아둔 채로 저장될 일이 없어요.

### 3. 프로젝트 워크벤치 — "엑셀 간트차트, 진짜 작별합니다"

`프로젝트_일정_v7_최종_진짜최종.xlsx` 같은 파일, 한 번쯤 만들어 보셨죠. 팀웍스는 프로젝트를 **2단계 구조**로 정리해서 누가 봐도 한눈에 흐름이 들어오게 합니다.

- **프로젝트** — 업무의 가장 큰 틀. "2분기 신제품 런칭" 같은 단위로 만들고, "기획 → 디자인 → 개발 → QA → 출시" 같은 단계를 직접 정할 수 있어요.
- **프로젝트 일정** — 프로젝트 안의 구체적인 업무 덩어리. 간트차트의 한 줄로 표시되며, 담당자·기간·진행률·지연 여부를 입력합니다.
- **세부 일정** — 큰 업무를 더 잘게 쪼갠 할 일. "오늘 이거 하나만 끝내자" 같은 목표 세우기에 딱.
- **자동 시각화** — 진행률은 막대 안에 채워지고, 지연된 일정은 색이 바뀌어 강조됩니다. 팀장도 팀원도 "여기 빨리 처리해야겠다" 바로 알아챌 수 있어요.
- **만든 사람이 책임** — 누구나 만들 수 있지만 수정·삭제는 만든 사람만. 팀장이라도 남의 작업을 임의로 손대지 못하는 안전한 구조입니다.

### 4. 팀 채팅 + 자료실 — "업무 대화와 사담, 그리고 자료까지 깔끔하게 정리"

카톡에 업무 얘기가 끼면 "아까 그 파일 어디 있더라?" 영원히 못 찾는 경험, 다들 해보셨죠. 팀웍스 채팅은 **목적별로** 분리됩니다.

**일자별 팀 채팅** — 캘린더에서 날짜를 클릭하면 그 날의 메시지로 자동 전환. 일정과 대화의 맥락이 한 화면에서 이어집니다.

- **일반 대화** — 점심 메뉴, 회의 후기, 잡담. 팀원 전부 볼 수 있어요.
- **업무보고** — 팀장에게 올리는 보고. 기본은 팀장만 보고, 팀장이 "이건 다 같이 봐도 돼요"라고 허락한 사람만 추가 열람. 동료가 훔쳐볼 수 없는 안전한 구조예요.
- **공지사항** — "내일 오전 10시 전체 회의!" 같은 알림, 채팅 맨 위에 고정. 팀원 누구나 올릴 수 있고, 지우는 건 올린 본인이나 팀장만.

**프로젝트 전용 채팅방** — 프로젝트마다 독립된 채팅방이 자동으로 따라옵니다. "신규 대시보드" 프로젝트를 열면 그 프로젝트만의 메시지·공지가 따로 정리되어, 일자별 팀 채팅과 절대 섞이지 않아요.

**자료실 (게시판)** — 회의록, 보고서, 자료, 이미지를 채팅방 sub-탭에서 한 곳에 정리합니다.

- 글 제목 + 본문 + 첨부파일(최대 10MB) 한 번에 등록. 카톡에서 며칠만 지나면 사라지는 파일 걱정 끝.
- 팀원 누구나 글을 쓸 수 있고, 수정·삭제는 작성자 본인만.
- 채팅과 마찬가지로 일자별 자료실 / 프로젝트 자료실로 깔끔히 분리됩니다.
- 보안 검증 통과한 안전한 형식(이미지·PDF·docx·xlsx·pptx·zip 등)만 업로드 가능. 악성 파일 자동 차단.

### 5. AI 버틀러 "찰떡이" — "한 줄 적으면 알아서 다 합니다"

매뉴얼 뒤지고, 단톡방에서 동료 호출하고, 일정 등록 버튼 여기저기 누르던 시간. 이제는 **한 줄**만 적으세요.

찰떡이는 우측 채팅 영역의 **AI 버틀러** 탭에 있는 단일 입력창에서 동작합니다. 모드를 직접 고를 필요가 **전혀 없어요** — 자연어로 묻거나 부탁하면 찰떡이가 알아서 6가지 길 중 적절한 곳으로 보냅니다.

- **사용법 질문** — "포스트잇 색깔 종류 알려줘", "프로젝트 등록하는 법" — 팀웍스 공식 문서를 직접 검색해 답변. **공식 문서 N건 참조** 출처가 함께 표시되니 신뢰도 보장.
- **일반 질문** — "오늘 서울 날씨 어때?", "오늘 뉴스 검색해줘" — 시사·날씨·일반 지식까지 웹 검색 결과를 가져와 답합니다.
- **일정 조회** — "오늘 일정 알려줘", "디자인 리뷰 언제야?" — 캘린더 들어갈 필요 없이 자연어로 즉시 조회.
- **일정 등록** — "내일 오후 3시 주간회의 등록해줘" — 한 줄로 일정 등록 끝. 단, 등록 전에 "**이렇게 등록할까요?** 미리보기" 카드를 먼저 띄우고 ✓ 클릭해야 실제로 들어갑니다.
- **일정 삭제** — "내일 회의 삭제해줘", "5월 14일 고객미팅 지워줘" — 대상 일정을 식별해 미리보기 → ✓ 후 삭제. 후보가 여럿이면 시각·제목으로 좁히기 안내. **"전체 삭제" 같은 일괄 시도는 자동 차단** — 한 번에 하나만.
- **일정 수정** — "내일 회의 수정해줘" — 식별 → 새 일시 → 새 제목 차례로 묻고 변경 전/후 비교 미리보기 → ✓ 후 반영. 바꾸지 않을 항목은 **"그대로"** 라고만 답하면 부분 수정 가능.
- **다중 턴 대화** — "내일 회의 등록해줘" 같이 시간이 빠진 요청에는 "몇 시에 잡을까요?" 라고 되묻습니다. "오후 3시" 라고 답하면 그제서야 미리보기가 떠요.

**찰떡이가 안 하는 일** — 일정 **취소**(삭제와 의미 모호), 일정 **색깔·설명·메모 변경**(캘린더 화면에서 직접), 프로젝트·자료실·채팅 같은 다른 영역의 변경 작업은 정중히 거절합니다. 또한 **다른 사람이 만든 일정은 변경 불가** — 화면이든 찰떡이든 생성자 본인만 수정·삭제할 수 있습니다. 안전이 우선입니다.

> "개떡같이 물어도 **찰떡**같이 대답하라고 주인님이 지어주셨어요~~" — 그래서 이름이 찰떡이입니다.
