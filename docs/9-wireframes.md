## 화면 흐름도 (Screen Flow)

```
[비인증 상태]
      |
      +---> S-01 로그인 -----> (로그인 성공) -----> S-03 팀 목록
      |                                                  |
      +---> S-02 회원가입 --> (가입 성공) ---------> S-03 팀 목록
                                                         |
                               +-------------------------+-------------------------+
                               |                         |                         |
                         [팀 생성 버튼]             [팀 카드 클릭]          [팀 탐색 버튼]
                               |                         |                         |
                           S-04 팀 생성            S-05 팀 메인            S-04B 팀 공개 목록
                               |                  (캘린더 + 채팅)                 |
                               |                    /   |   \              (가입 신청)
                               +----> S-05       /     |     \               |
                                           일정클릭  나의할일  날짜클립      |
                                               |       |       |            |
                                           S-06   S-04C     채팅탭    가입 신청 → 팀장 승인 대기
                                                (나의 할 일)
                                                  |
                                            [승인/거절]

권한에 따른 분기:
  LEADER: S-05, S-06(생성/수정/삭제), S-04C(나의 할 일) 모두 접근 가능
  MEMBER: S-05(조회/채팅만), S-06(조회만, 수정버튼 숨김), S-04B(팀 탐색) 접근 가능
```

## S-01 로그인

**화면 ID**: S-01
**화면명**: 로그인
**경로**: `/login`
**접근 권한**: 비인증 (인증된 사용자 접근 시 `/`로 리다이렉트)

### 주요 컴포넌트

- 서비스 로고 / 타이틀
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼 (입력값 유효할 때 활성화)
- 에러 메시지 영역
- 회원가입 페이지 링크

### 와이어프레임 (모바일 375px)

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│         Team CalTalk                │  ← 서비스 로고/타이틀 (중앙 정렬)
│        팀 캘린더 + 채팅              │  ← 서브타이틀
│                                     │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  이메일                      │   │  ← 이메일 입력 필드
│  │  user@example.com           │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  비밀번호                    │   │  ← 비밀번호 입력 필드 (마스킹)
│  │  ••••••••                   │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │         로그인               │   │  ← Primary 버튼 (활성/비활성 상태)
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  ⚠ 이메일 또는 비밀번호가    │   │  ← 에러 메시지 (오류 시에만 표시)
│  │    올바르지 않습니다.        │   │
│  └─────────────────────────────┘   │
│                                     │
│   계정이 없으신가요?  [회원가입 →]   │  ← 회원가입 링크
│                                     │
└─────────────────────────────────────┘
```

### 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 이메일 / 비밀번호 입력 | 실시간 클라이언트 유효성 검증, 형식 오류 시 인라인 메시지 표시 |
| 로그인 버튼 클릭 | `POST /api/auth/login` 요청, 성공 시 `/`로 리다이렉트 |
| API 401 응답 | 에러 메시지 영역에 "이메일 또는 비밀번호가 올바르지 않습니다." 표시 |
| 회원가입 링크 클릭 | `/signup` (S-02)으로 이동 |

## S-02 회원가입

**화면 ID**: S-02
**화면명**: 회원가입
**경로**: `/signup`
**접근 권한**: 비인증 (인증된 사용자 접근 시 `/`로 리다이렉트)

### 주요 컴포넌트

- 서비스 로고 / 타이틀
- 이름 입력 필드 (최대 50자)
- 이메일 입력 필드
- 비밀번호 입력 필드
- 회원가입 버튼
- 에러 메시지 영역
- 로그인 페이지 링크

### 와이어프레임 (모바일 375px)

```
┌─────────────────────────────────────┐
│                                     │
│         Team CalTalk                │  ← 서비스 로고/타이틀
│           회원가입                   │  ← 화면 타이틀
│                                     │
│  ┌─────────────────────────────┐   │
│  │  이름                        │   │  ← 이름 입력 (최대 50자)
│  │  홍길동                      │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  이메일                      │   │  ← 이메일 입력
│  │  user@example.com           │   │
│  └─────────────────────────────┘   │
│  ✓ 사용 가능한 이메일입니다.         │  ← 인라인 검증 메시지 (상황별)
│                                     │
│  ┌─────────────────────────────┐   │
│  │  비밀번호                    │   │  ← 비밀번호 입력 (마스킹)
│  │  ••••••••                   │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │          회원가입             │   │  ← Primary 버튼
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  ⚠ 이미 사용 중인 이메일입니다│   │  ← 에러 메시지 (오류 시에만 표시)
│  └─────────────────────────────┘   │
│                                     │
│   이미 계정이 있으신가요? [로그인 →] │  ← 로그인 링크

```

### 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 필드 입력 중 | 실시간 클라이언트 유효성 검증 (이메일 형식, 이름 길이 등) |
| 회원가입 버튼 클릭 | `POST /api/auth/signup` 요청, 성공 시 `/`로 리다이렉트 |
| API 409 응답 | "이미 사용 중인 이메일입니다." 에러 표시 |
| 로그인 링크 클릭 | `/login` (S-01)으로 이동 |

## S-03 팀 목록 (홈)

**화면 ID**: S-03
**화면명**: 팀 목록
**경로**: `/`
**접근 권한**: 인증 필요 (비인증 시 `/login`으로 리다이렉트)

### 주요 컴포넌트

- 상단 네비게이션 (사용자 이름, 로그아웃 버튼)
- 팀 목록 (팀명, 내 역할 배지, 생성일)
- 팀 생성 버튼 (FAB 또는 상단 버튼)
- 팀 탐색 버튼 (공개 팀 목록 보기)
- 빈 상태 안내 (팀이 없을 때)

### 와이어프레임 (모바일 375px)

```
┌─────────────────────────────────────┐
│  Team CalTalk         홍길동 [로그아웃] │  ← 상단 네비게이션
├─────────────────────────────────────┤
│                                     │
│  내 팀 목록                          │  ← 섹션 타이틀
│                                     │
│  ┌─────────────────────────────┐   │
│  │  개발팀                      │   │  ← 팀 카드
│  │  [LEADER]   2026-04-01      │   │    (역할 배지, 생성일)
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  디자인팀                    │   │  ← 팀 카드
│  │  [MEMBER]   2026-04-03      │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  마케팅팀                    │   │  ← 팀 카드
│  │  [MEMBER]   2026-04-05      │   │
│  └─────────────────────────────┘   │
│                                     │
│                                     │
│                        [+ 팀 생성]  │  ← FAB 버튼 (우측 하단)
└─────────────────────────────────────┘

--- 빈 상태 (팀이 없을 때) ---

┌─────────────────────────────────────┐
│  Team CalTalk         홍길동 [로그아웃] │
├─────────────────────────────────────┤
│                                     │
│                                     │
│           아직 팀이 없습니다.         │  ← 빈 상태 안내
│       팀을 생성하거나 초대를 기다려    │
│                주세요.               │
│                                     │
│       ┌───────────────────────┐     │
│       │      + 팀 생성         │     │  ← CTA 버튼
│       └───────────────────────┘     │
│                                     │
└─────────────────────────────────────┘
```

### 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 화면 진입 | `GET /api/teams` 요청으로 팀 목록 로드 |
| 팀 카드 클릭 | `/teams/[teamId]` (S-05)로 이동 |
| 팀 생성 버튼 클릭 | `/teams/new` (S-04)로 이동 |
| 팀 탐색 버튼 클릭 | `/teams/explore` (S-04B)로 이동 |
| 로그아웃 버튼 클릭 | 클라이언트 토큰 삭제 후 `/login`으로 이동 |

## S-04 팀 생성

**화면 ID**: S-04
**화면명**: 팀 생성
**경로**: `/teams/new`
**접근 권한**: 인증 필요

### 주요 컴포넌트

- 뒤로가기 버튼
- 팀명 입력 필드 (최대 100자)
- 생성 버튼
- 에러 메시지 영역

### 와이어프레임 (모바일 375px)

```
┌─────────────────────────────────────┐
│  [← 뒤로]        팀 생성             │  ← 상단 헤더
├─────────────────────────────────────┤
│                                     │
│                                     │
│  새 팀을 만들어보세요.                │  ← 안내 문구
│  생성 후 팀원을 초대할 수 있습니다.   │
│                                     │
│  팀 이름 *                           │  ← 필드 레이블 (* 필수)
│  ┌─────────────────────────────┐   │
│  │  팀 이름을 입력하세요         │   │  ← 팀명 입력 (placeholder)
│  └─────────────────────────────┘   │
│  0 / 100자                          │  ← 글자 수 카운터
│                                     │
│  ⚠ 팀 이름은 필수입니다.             │  ← 에러 메시지 (오류 시)
│                                     │
│                                     │
│  ┌─────────────────────────────┐   │
│  │           팀 생성            │   │  ← Primary 버튼
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

### 인터랙션

| 이벤트 | 동작 |
|--------|------|
| 팀 이름 입력 | 글자 수 실시간 카운터 갱신, 빈값이면 버튼 비활성화 |
| 생성 버튼 클릭 | `POST /api/teams` 요청, 성공 시 `/teams/[teamId]` (S-05)로 리다이렉트 |
| API 오류 | 에러 메시지 표시 |
| 뒤로가기 버튼 | `/` (S-03)으로 이동 |

## S-05 팀 메인 (캘린더 + 채팅)

**화면 ID**: S-05
**화면명**: 팀 메인
**경로**: `/teams/[teamId]`
**접근 권한**: 해당 팀의 LEADER 또는 MEMBER

### 주요 컴포넌트

- 상단 네비게이션 (팀명, 팀 설정 메뉴)
- 캘린더 뷰 (월/주/일 탭 전환)
- 이전/다음 월 네비게이션
- 일정 카드 (캘린더 날짜 셀 내)
- 채팅 영역 (날짜 헤더, 메시지 목록)
- 메시지 입력창 + 전송 버튼
- 일정 변경 요청 버튼 (MEMBER 전용)
- 일정 추가 버튼 (LEADER 전용)
