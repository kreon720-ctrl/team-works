Q2

주소창에 teamworks.my 치면 구글 검색으로 가요

앞에 https:// 를 붙여주세요 → https://teamworks.my

.my 도메인은 .com 처럼 익숙하지 않아 주소가 아닌 검색어로 인식되는 경우가 있어요. 크롬·엣지의 주소+검색 통합창(omnibox) 때문입니다.

✗ 안 됨: ✓ 됨: ┌──────────────────┐ ┌──────────────────────────┐ │ teamworks.my │ │ https://teamworks.my │ └──────────────────┘ └──────────────────────────┘ ↓ Enter ↓ Enter 구글 검색결과로 이동 정상 접속!

한 번만 접속하면 다음부터는 편해요

  • 첫 접속 후 북마크(즐겨찾기)에 저장하면 클릭 한 번
  • 또는 브라우저 시작 페이지로 설정
  • 모바일에서는 홈 화면에 추가(앱처럼 사용)
TIP 한 번 https:// 붙여 정상 접속하면, 다음부터는 teamw 만 쳐도 자동완성으로 뜹니다.
Q3

음성 입력이 중간에 끊겨요

마이크를 다시 눌러 이어서 말하면 자동으로 붙어요.

휴대폰 음성 입력은 짧은 침묵에도 자동으로 멈추는 경우가 있는데요, 끊겨도 괜찮습니다. 마이크 버튼을 다시 누르고 이어 말하면 기존 텍스트 뒤에 자동으로 이어 붙습니다.

1) 🎤 클릭 → "오늘 오후 3시" → (자동 멈춤) 입력창: "오늘 오후 3시" 2) 🎤 다시 클릭 → "주간회의 등록해줘" → (자동 멈춤) 입력창: "오늘 오후 3시 주간회의 등록해줘" ↑ 자동 이어붙음 3) 검토 → [전송]

처음부터 다시 말하고 싶을 땐? 입력창 우측의 작은 ✕ (클리어 버튼) 을 누르면 비워집니다.

TIP 한 번에 길게 말하기 부담스러우면 짧게 끊어서 여러 번 누르세요. 결과는 한 줄로 자연스럽게 합쳐집니다.
Q4

어떤 인터넷 브라우저를 써야 하나요?

대부분 지원 — 디바이스별 권장 브라우저만 알아두면 됩니다.
디바이스권장
안드로이드Chrome ⭐
아이폰·아이패드Safari ⭐
윈도우 PCChrome / Edge
맥Safari / Chrome
리눅스Chrome / Firefox

특히 음성 입력은 OS 가 제공하는 음성 엔진(Apple/Google) 을 브라우저가 위임 호출하는 구조라서, 디바이스 native 브라우저가 가장 정확합니다.

TIP 다른 브라우저(Firefox, Brave 등)를 쓰셔도 텍스트 입력·일정 관리 같은 기본 기능은 모두 동작합니다. 음성 정확도 차이만 있을 뿐.
Q5

AI 찰떡이가 일정 명령을 잘못 알아들어요

[시점] + [목적어] + [동사] 를 명확히 — 한 줄에 한 가지 의도만.

찰떡이는 자연어를 잘 이해하지만, 너무 짧거나 모호한 표현은 조회인지 등록인지 헷갈려 잘못 분류할 수 있어요.

✗ 잘못된 예시

  • "일정 등록 조회" — 등록인지 조회인지 동시 의도?
  • "회의" — 시점·동사 모두 없음
  • "내일" — 동사 없음

✓ 잘 인식되는 패턴

시점 목적어 동사 ───────── ────────── ────────── 오늘 점심 약속 조회해줘 내일 오후 회의 일정 삭제해줘 5월 22일 디자인 리뷰 등록해줘 다음 주 모든 회의 보여줘

의도별 동사 키워드

의도키워드
조회알려줘 / 보여줘 / 조회 / 확인
등록등록 / 추가 / 잡아줘 / 생성
삭제삭제 / 지워 / 제거
수정수정 / 변경 / 바꿔 / 옮겨

추가 자주 묻는 질문 (10개)
A1

팀은 어떻게 만들고, 어떻게 가입해요?

직접 만들면 자동으로 팀장 — 가입은 팀 찾기에서 검색 후 신청.

새 팀 만들기

  • 좌측 상단 햄버거 메뉴(≡) → 새 팀 만들기
  • 팀 이름·설명·공개/비공개 선택
  • 만들면 자동으로 팀장

다른 팀 가입: 좌측 메뉴 → 팀 찾기 → 공개 팀 선택 → 가입 신청 → 팀장이 승인하면 접근 가능

TIP 회사 팀 + 사이드 프로젝트 + 동호회까지 한 계정으로 동시 운영 가능. 팀별로 캘린더·채팅·자료가 따로 보관돼요.
A2

'일정'과 '포스트잇' 차이가 뭐예요?

시간 약속이면 일정 / 가벼운 메모면 포스트잇.
구분시간등록
일정시·분 단위+ 버튼 또는 찰떡이
포스트잇날짜만색상 → 날짜 탭

예) "내일 점심 12시 김부장님과 식사" → 일정 · "내일 점심 사가기" → 포스트잇

A3

다른 사람 일정·포스트잇을 제가 지울 수도 있나요?

아니요 — 만든 사람 본인만 수정·삭제할 수 있습니다.
  • 올리기·보기: 팀원이면 누구나
  • 수정·삭제: 작성자 본인만
  • 팀장이라도 다른 사람 일정은 못 지움 (팀장 권한은 팀 운영 한정)
A4

AI 찰떡이한테 뭘 시킬 수 있어요?

일정 관리 + 사용법 질문 + 일반 검색까지.
의도예시
조회"오늘 일정 알려줘"
등록"내일 오후 3시 회의 등록"
삭제"내일 회의 삭제"
수정"내일 회의 수정"
사용법"포스트잇 색 종류"
웹 검색"오늘 뉴스"

등록·삭제·수정은 항상 미리보기 카드 + ✓ / ✕ 가 한 번 뜹니다. ✓ 눌러야 진짜 반영.

A5

마이크 버튼은 어떻게 써요?

한 번 탭 → 말하기 → 자동 멈춤 → [전송].
  • 모바일 AI 찰떡이 와 팀 채팅 입력창 옆 🎤 버튼
  • PC 에서는 키보드가 빠르니 모바일에서만 노출

인식이 잘 안 될 때: 마이크 권한 확인 → 조용한 곳에서 재시도 → 키보드로 부분 수정. 끊겼을 땐 Q3 참고.

A6

다크 모드 / 라이트 모드는 어떻게 바꾸나요?

우측 상단 토글 스위치 한 번 클릭.
┌────────────────────────────────┐ │ 기획팀장 [3] ⏻ ⚪⚫ ⋮ │ └────────────────────────────────┘ ↑ 밝은 모드 ↔ 어두운 모드

선택은 브라우저별로 기억 되며 새로고침 후에도 유지됩니다. 일정 색상도 두 모드 모두에서 잘 보이게 자동 보정.

A7

종료 시각 없이도 일정 등록할 수 있나요?

네 — 시작만 정해두고 종료는 비워둬도 됩니다.
제목 * : 회식 시작 일시 * : 2026-05-22 19:00 종료 일시 (선택): [비워둠] ← 선택 입력 ↓ [등록]

표시 — MonthView 칩: 19:00 회식 (~ 없음) · WeekView/DayView: 텍스트 분량만큼 작은 카드 · AI 답변: 5. 22. 19:00 회식

A8

가입 신청 승인은 어디서 하나요?

좌측 메뉴 나의 할 일 에서 한 번에.
  • 여러 팀의 팀장이라도 모든 가입 신청이 한 곳에 모임
  • 헤더 상단 빨간 배지 숫자 로 새 신청 알림
  • 한 번에 여러 신청 승인 가능
TIP 비공개 팀에는 가입 신청 자체가 안 와요 (검색 안 됨).
A9

한 일정이 여러 날에 걸쳐도 등록 가능한가요?

네 — 시작·종료를 다른 날짜로 설정하면 됩니다.
제목 : 부산 출장 시작 일시 *: 2026-05-15 09:00 종료 일시 : 2026-05-19 18:00 ← 다른 날
  • MonthView: 5/15~5/19 까지 가로 색상 띠
  • WeekView: 상단 "종일·다일" 영역에 한 줄
  • AI 답변: 5. 15. 09:00 ~ 5. 19. 18:00 부산 출장

AI 찰떡이로도 "5월 15일부터 19일까지 부산 출장 등록" 처럼 자연어 가능.

A10

모바일에서 캘린더·프로젝트·팀채팅·AI 찰떡이는 어떻게 전환하나요?

화면 상단 4개 탭 으로 전환.
┌──────┬──────┬──────┬──────┐ │ 📅 │ 📋 │ 💬 │ ✨ │ │캘린더│프로젝트│팀채팅│찰떡이│ └──────┴──────┴──────┴──────┘
  • 캘린더: 월·주·일 + 포스트잇
  • 프로젝트: 갠트 차트 + 세부일정 (가로 스크롤)
  • 팀채팅: 일자/프로젝트별 채팅 + 자료실
  • AI 찰떡이: 자연어 명령 + 사용법 + 웹 검색

탭을 누르는 대신 좌우 스와이프 로도 전환 가능. AI 찰떡이 입력창 우측 📅 캘린더 아이콘을 누르면 화면 위쪽에 캘린더가 함께 떠서 답변 결과를 즉시 확인할 수 있습니다.

TIP PC 에서는 캘린더 + 팀채팅이 좌우로 동시에 표시되고, AI 찰떡이는 우측 상단 탭으로 전환.
