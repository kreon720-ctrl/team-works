여러 앱 왔다 갔다, 이제 끝

슬랙에서 일정 얘기하고, 구글 캘린더에 옮기고, 노션에 정리하고, 엑셀로 간트차트 그리고, 단톡방에서 자료 받고… 하루에 몇 번이나 반복하셨어요? 팀웍스 한 탭만 켜두면 일정·프로젝트·채팅·자료까지 모든 흐름이 한 화면 안에서 굴러갑니다.

팀 빌딩부터 보고까지, 원스톱

새 팀 꾸리기 → 팀원 모으기 → 일정 짜기 → 프로젝트 진행 → 보고까지. 팀이 시작하고 끝낼 때까지 필요한 모든 단계가 한 앱 안에 있어요.

복잡한 업무 흐름, 한 장으로 정리

2단계 프로젝트 뷰로 큰 그림과 디테일을 동시에 봅니다. 진행률은 %로, 늦어진 일은 색이 바뀌어 자동으로 "여기 빨간불"이라고 알려줘요. 프로젝트_일정_v7_최종_진짜최종.xlsx 만들던 야근, 작별 인사 드릴 시간.

지난주 결정사항? 1초 컷

회의·메모·대화가 날짜별로 자동 정리됩니다. 며칠 전 결정사항을 다시 꺼내거나 주간 회고할 때 단톡방 스크롤 무한히 내릴 필요 없이 날짜만 찍으면 끝. 업무처리에 필요한 자료공유와 대화를 한화면에서.

당신 곁의 AI 비서 찰떡이

"내일 오후 3시 주간회의 등록해줘" / "5월 14일 회의 삭제해줘" / "내일 회의 수정해줘" — 한 줄이면 AI 비서 찰떡이가 일정을 잡고·지우고·바꿔드립니다. 등록·삭제·수정 모두 미리보기 카드 → ✓ 클릭 후에만 실제로 반영되니 실수도 없어요. 사용법이 헷갈릴 땐 그냥 편안하게 물어보세요. 개떡같이 물어도 찰떡같이 답해줍니다.

휴대폰 키보드 답답할 땐, 그냥 말하세요

휴대폰 작은 키보드로 오타 내며 다시 지우고, 자동완성에 엉뚱한 단어 떠서 또 고치고… 답답하셨죠. 입력창 옆 마이크 버튼 한 번이면 일정 등록부터 팀 메시지 전송까지 말로 끝납니다. 인식한 텍스트는 입력창에 채워두니, 검토하고 [전송] 버튼 누르면 끝.

I. 팀 빌딩 — "내 팀, 내가 만들고 운영해요"

새 프로젝트 시작할 때마다 IT팀에 "채널 만들어 주세요" 부탁하거나, 단톡방 새로 파서 사람 초대하느라 진땀 빼던 경험 있으시죠. 팀웍스는 내가 직접 팀을 만들고, 만든 사람이 자동으로 팀장이 됩니다.

  • 자유로운 팀 구성 — 팀명·설명을 정하고, 공개로 둘지 비공개로 둘지도 직접 선택. 공개 팀이면 다른 사람들이 직접 가입 신청을 해와요. 비공개로 두면 개인 일정 관리도 가능합니다.
  • 승인은 한 곳에서 — 여러 팀을 이끌고 있어도 '나의 할 일' 화면에 가입 신청이 전부 모여, 한 번에 받을 사람만 골라 승인하면 끝.
  • 운영 권한도 팀장 손에 — 팀명·설명·공개 여부는 언제든 수정. 정중히 작별할 팀원이 있다면 명단에서 빼는 것도 가능합니다.
  • 한 사람, 여러 팀 — 한 계정으로 회사 팀 + 사이드 프로젝트 팀 + 동호회 팀까지 동시에. 어떤 팀에서는 팀장, 다른 팀에서는 팀원도 자유롭게.

II. 스마트 캘린더 — "팀 일정, 더 이상 헷갈리지 않아요"

"다음 주 수요일 회의 몇 시였지?" 단톡방 스크롤 끝까지 내려본 적, 다들 한 번쯤 있죠.

  • 월·주·일 뷰 — 이번 달 전체 그림은 월 뷰로, 이번 주 시간표는 주 뷰로, 오늘 빡빡한 일정은 일 뷰로. 보고 싶은 대로 골라 보세요.
  • 누구나 등록, 본인만 수정·삭제 — 팀원이라면 누구나 일정을 추가할 수 있고, 내가 올린 일정만 내가 수정·삭제할 수 있어요. 누가 마음대로 회의 시간을 지워버릴 걱정 없음. 화면이든 AI 찰떡이든 동일하게 생성자 본인만 변경 가능합니다.
  • 5가지 색상으로 구분 — indigo, blue, emerald, amber, rose. "회의는 indigo, 마감은 rose" 같은 팀 규약을 정해두면 캘린더만 봐도 뭐가 뭔지 바로 보입니다.
  • 컬러 포스트잇 — "내일 거래처 미팅 전에 자료 챙기기!" 같은 가벼운 메모, 색깔 골라서 날짜에 척 붙여둘 수 있어요. 시간 정해진 일정은 일정으로, 그날 챙길 잡일은 포스트잇으로.

III. 프로젝트 워크벤치 — "엑셀 간트차트, 진짜 작별합니다"

프로젝트_일정_v7_최종_진짜최종.xlsx 같은 파일, 한 번쯤 만들어 보셨죠. 팀웍스는 프로젝트를 2단계 구조로 정리해서 누가 봐도 한눈에 흐름이 들어오게 합니다.

  • 프로젝트 — 업무의 가장 큰 틀. "2분기 신제품 런칭" 같은 단위로 만들고, "기획 → 디자인 → 개발 → QA → 출시" 같은 단계를 직접 정할 수 있어요.
  • 프로젝트 일정 — 프로젝트 안의 구체적인 업무 덩어리. 간트차트의 한 줄로 표시되며, 담당자·기간·진행률·지연 여부를 입력합니다.
  • 세부 일정 — 큰 업무를 더 잘게 쪼갠 할 일. "오늘 이거 하나만 끝내자" 같은 목표 세우기에 딱.
  • 자동 시각화 — 진행률은 막대 안에 채워지고, 지연된 일정은 색이 바뀌어 강조됩니다. 팀장도 팀원도 바로 알아챌 수 있어요.

IV. 팀 채팅 + 자료실 — "업무 대화와 사담, 그리고 자료까지 깔끔하게"

카톡에 업무 얘기가 끼면 "아까 그 파일 어디 있더라?" 영원히 못 찾는 경험, 다들 해보셨죠. 팀웍스 채팅은 목적별로 분리됩니다.

일자별 팀 채팅

캘린더에서 날짜를 클릭하면 그 날의 메시지로 자동 전환. 일정과 대화의 맥락이 한 화면에서 이어집니다.

  • 일반 대화 — 점심 메뉴, 회의 후기, 잡담. 팀원 전부 볼 수 있어요.
  • 업무보고 — 팀장에게 올리는 보고. 기본은 팀장만 보고, 팀장이 허락한 사람만 추가 열람. 동료가 훔쳐볼 수 없는 안전한 구조예요.
  • 공지사항 — "내일 오전 10시 전체 회의!" 같은 알림, 채팅 맨 위에 고정. 팀원 누구나 올릴 수 있고, 지우는 건 올린 본인이나 팀장만.

자료실 (게시판)

회의록, 보고서, 자료, 이미지를 채팅방 sub-탭에서 한 곳에 정리합니다.

  • 글 제목 + 본문 + 첨부파일(최대 10MB) 한 번에 등록. 카톡에서 며칠만 지나면 사라지는 파일 걱정 끝.
  • 팀원 누구나 글을 쓸 수 있고, 수정·삭제는 작성자 본인만.
  • 채팅과 마찬가지로 일자별 자료실 / 프로젝트 자료실로 깔끔히 분리됩니다.
  • 보안 검증을 통과한 안전한 형식(이미지·PDF·docx·xlsx·pptx·zip 등)만 업로드 가능. 악성 파일 자동 차단.

V. 당신 곁의 AI 비서 찰떡이 — "개떡같이 물어도 찰떡같이 답해줍니다."

찰떡이는 6가지 기능을 제공합니다.

  • 사용법 질문 — "포스트잇 색깔 종류 알려줘", "프로젝트 등록하는 법" — 팀웍스 공식 문서를 직접 검색해 답변.
  • 일반 질문 — "오늘 서울 날씨 어때?", "오늘 뉴스 검색해줘" — 시사·날씨·일반 지식까지 웹 검색 결과를 가져와 답합니다.
  • 일정 조회 — "오늘 일정 알려줘", "디자인 리뷰 언제야?" — 캘린더 들어갈 필요 없이 자연어로 즉시 조회.
  • 일정 등록 — "내일 오후 3시 주간회의 등록해줘", "13일 11시 반 직원점심 잡아줘" — 한 줄이면 일정 등록 끝. 등록 전에 "이렇게 등록할까요?" 미리보기 카드를 먼저 띄우고 ✓ 클릭해야 실제로 들어갑니다.
  • 일정 삭제 — "내일 회의 삭제해줘", "5월 14일 고객미팅 지워줘", "내일 약속 취소해줘" — 대상 일정을 식별해 미리보기 → ✓ 후 삭제.
  • 일정 수정 — "내일 회의 수정해줘" — 수정할 사항을 단계별로 묻고 답한 후 미리보기 → ✓ 후 반영.

다중 턴 대화 — "내일 회의 등록해줘" 같이 시간이 빠진 요청에는 "몇 시에 잡을까요?" 라고 되묻습니다. "오후 3시" 라고 답하면 그제서야 미리보기가 떠요. "5시"처럼 오전인지 오후인지 모호하면 "오전/오후 어느 쪽일까요?" 한 번 더 확인.

찰떡이가 안 하는 일 — 프로젝트·자료실·채팅 같은 다른 영역의 변경 작업은 정중히 거절합니다.

"개떡같이 물어도 찰떡같이 대답하라고 주인님이 지어주셨어요~~" — 그래서 이름이 찰떡이입니다.

VI. Speak & Go (STT) — "불편한 휴대폰 키보드 안녕"

휴대폰 키보드 작아서 답답할 때, 마이크 버튼 한 번이면 음성으로 끝납니다.

  • 마이크 버튼 한 번 — AI 찰떡이 입력창 옆과 팀채팅 입력창 옆에 마이크 아이콘이 있어요. 누르고 한국어로 말하면 텍스트가 입력창에 채워집니다. 검토 후 [전송] 클릭하면 완료.
  • 자동 침묵 감지 — 발화 후 잠시 멈추면 자동으로 변환 시작. 길게 누를 필요 없습니다.
  • 확인 후 전송 — 인식 결과는 입력창에 채워두기만 합니다. 오인식이 있어도 한 번 더 살펴보고 보낼 수 있어 안심.
