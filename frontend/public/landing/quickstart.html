1로그인부터 팀 가입 승인까지

새 계정을 만들고, 팀을 꾸리거나 기존 팀에 가입하는 전 과정.

회원가입 → 팀 만들기 → 팀 검색·가입 → 팀장 승인
1

회원가입 · 로그인

웹브라우저로 http://teamworks.my 접속 → 회원가입 화면에서 이메일·비밀번호 입력 후 가입. 이후 로그인하면 내 팀 목록 화면으로 이동합니다.

처음이라면 회원가입 → 링크로 이동:

2

내 팀 목록 화면

로그인 직후 만나는 첫 화면. 내가 속한 팀들이 카드로 나타납니다. 처음이면 비어 있고, 하단 두 버튼으로 시작합니다.

  • + 팀 생성 — 직접 새 팀을 만들고 자동으로 팀장이 됨
  • 팀 검색 — 기존 공개 팀에 가입 신청
3

+ 팀 생성 — 직접 팀 만들기

새 팀을 만들 때 입력:

  • 팀 이름 · 설명 (선택)
  • 공개 여부 — 공개(다른 사람이 검색 가능) / 비공개(나만)
TIP · 비공개 팀을 만들어 혼자 쓰면 개인 일정 관리 용도로 활용 가능합니다.
4

팀 검색 → 가입 신청

이미 만들어진 공개 팀에 들어가려면 팀 검색 버튼. 팀 이름으로 검색하고 가입 신청을 누르면 신청이 팀장에게 전달됩니다.

5

팀장의 가입 승인 (LEADER 전용)

내가 팀장인 팀에 가입 신청이 들어오면 카드 우측에 빨간 배지(숫자)가 표시됩니다. 클릭하면 나의 할 일 화면이 열리고, 신청자별로 승인·거절 버튼이 나옵니다.

TIP · 여러 팀을 이끌어도 모든 팀의 가입 신청이 한 화면에 모입니다.

2캘린더에서 일정 등록

팀 화면 → 캘린더에서 직접 입력하는 기본 방법.

팀 진입 → + 일정 등록 → 필드 입력 → 생성
1

팀 진입 → 캘린더

내 팀 목록에서 팀 카드를 누르면 팀 화면이 열립니다. 모바일에서는 상단 탭으로 캘린더 / 팀채팅 / AI 찰떡이를 전환합니다.

  • 월·주·일 뷰 전환
  • 이전·다음 화살표로 월 이동
  • 좌우 스와이프 도 가능 (모바일)
2

+ 일정 등록 모달

캘린더 상단의 노란색 + 일정 등록 버튼을 누르면 모달이 열립니다. 필수 항목:

  • 제목 (필수, 최대 200자)
  • 색상 — indigo · amber · blue · emerald · rose 중 선택
  • 설명 (선택)
  • 시작 일시 · 종료 일시 (필수, 종료 > 시작)
3

생성 → 캘린더 반영

하단 생성 버튼을 누르면 캘린더에 즉시 표시되고, 팀원 모두에게 보입니다. 내가 만든 일정만 내가 수정·삭제 가능합니다.

TIP · 캘린더 빈 날짜 셀을 직접 클릭해도 그 날짜로 등록 모달이 열립니다.

3찰떡이로 자연어 등록

버튼·필드 안 누르고 말 한 줄로 끝내기.

AI 찰떡이 탭 → 자연어 입력 → 미리보기 카드 → 승인 ✓
1

AI 찰떡이 탭 열기

팀 화면 상단의 AI 찰떡이 탭을 누르면 입력창과 예시 질문이 나옵니다.

2

자연어 입력 → 미리보기 카드

입력창에 일상 한국말로 적기:

  • "내일 오후 3시 주간회의 등록해줘"
  • "13일 11시 반 직원점심 잡아줘"
  • "5월 14일 회의 삭제해줘"

한 줄로 끝나면 곧바로 미리보기 카드가 표시됩니다. 정보가 부족하면 찰떡이가 되묻습니다 (예: "몇 시에 잡을까요?", "어떤 일정인가요?").

3

승인 ✓ → 등록 완료

미리보기 내용을 확인하고 승인 버튼을 눌러야 실제로 등록됩니다. 잘못 등록될 일이 없어요. 마음에 안 들면 취소.

TIP · 입력창 옆 마이크 버튼을 누르면 음성으로 입력할 수 있습니다 (Speak & Go). 휴대폰 키보드가 답답할 때 유용합니다.
4

찰떡이가 답할 수 있는 다른 질문들

  • 사용법 질문 — "포스트잇 색깔 종류 알려줘", "프로젝트 등록하는 법"
  • 일반 질문 — "오늘 서울 날씨 어때?", "오늘 뉴스 검색해줘"
  • 일정 조회 — "오늘 일정 알려줘", "내일 회의 언제야?"

개떡같이 물어도 찰떡같이 답해줍니다.
