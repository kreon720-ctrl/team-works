What's Inside

I. 팀 빌딩 — 내 팀, 내가 만들고 운영해요

새 프로젝트마다 IT팀에 "채널 만들어 주세요" 부탁하거나 단톡방 새로 파던 시간, 이제 끝. 내가 만들고, 만든 사람이 곧 팀장이에요.

  • 자유로운 팀 구성 — 팀명·설명·공개 여부를 직접 선택. 공개 팀이면 다른 사람이 직접 가입 신청을 보냅니다. 비공개로 두면 개인 일정 관리도 가능합니다.
  • 승인은 한 곳에서 — '나의 할 일' 화면에 모든 팀의 가입 신청이 모여, 한 번에 골라 승인.
  • 운영 권한도 팀장 손에 — 팀명·설명·공개 여부 수정, 팀원 강퇴까지 가능합니다.
  • 한 사람, 여러 팀 — 한 계정으로 회사 + 사이드 프로젝트 + 동호회까지 동시에.
II. 스마트 캘린더 — 팀 일정, 더 이상 헷갈리지 않아요

"다음 주 수요일 회의 몇 시였지?" 단톡방 스크롤 끝까지 내려본 적, 다들 한 번쯤 있죠.

  • 월·주·일 뷰 — 보고 싶은 단위로 골라 보세요.
  • 누구나 등록, 본인만 수정·삭제 — 내가 올린 일정만 내가 수정·삭제. 화면이든 AI 찰떡이든 동일하게 생성자 본인만 변경 가능.
  • 5가지 색상 구분 — indigo, blue, emerald, amber, rose. "회의는 indigo, 마감은 rose" 같은 팀 규약을 정해두면 한눈에.
  • 컬러 포스트잇 — "내일 거래처 미팅 전에 자료 챙기기!" 같은 메모, 색깔 골라 날짜에 척.
III. 프로젝트 워크벤치 — 엑셀 간트차트, 진짜 작별

프로젝트_일정_v7_최종_진짜최종.xlsx 같은 파일, 한 번쯤 만들어 보셨죠. 팀웍스는 프로젝트를 2단계 구조로 정리합니다.

  • 프로젝트 — 업무의 가장 큰 틀. "2분기 신제품 런칭" 같은 단위.
  • 프로젝트 일정 — 간트차트 한 줄. 담당자·기간·진행률·지연 여부 입력.
  • 세부 일정 — 더 잘게 쪼갠 할 일. "오늘 이거 하나만 끝내자".
  • 자동 시각화 — 진행률은 막대, 지연은 색으로. 빨간불 즉시 인지.
IV. 팀 채팅 + 자료실 — 대화·사담·자료를 깔끔히

카톡에 업무 얘기가 끼면 "아까 그 파일 어디 있더라?" 영원히 못 찾던 경험. 팀웍스 채팅은 목적별로 분리됩니다.

일자별 팀 채팅

캘린더에서 날짜를 클릭하면 그 날의 메시지로 자동 전환. 일정과 대화의 맥락이 한 화면에서 이어집니다.

  • 일반 대화 — 점심 메뉴, 회의 후기, 잡담. 팀원 전부 열람.
  • 업무보고 — 기본은 팀장만, 팀장이 허락한 사람만 추가 열람.
  • 공지사항 — 채팅 맨 위에 고정. 누구나 등록, 작성자·팀장만 삭제.

프로젝트 전용 채팅방

프로젝트마다 독립된 채팅방이 자동으로 따라옵니다. 일자별 팀 채팅과 절대 섞이지 않아요.

자료실 (게시판)

  • 제목 + 본문 + 첨부파일(최대 10MB) 한 번에. 카톡에서 사라지는 파일 걱정 끝.
  • 누구나 작성, 수정·삭제는 작성자 본인만.
  • 일자별 자료실 / 프로젝트 자료실로 분리.
  • 안전한 형식(이미지·PDF·docx·xlsx·pptx·zip 등)만 업로드 가능. 악성 파일 자동 차단.
V. 당신 곁의 AI 비서 찰떡이 — "개떡같이 물어도 찰떡같이 답해줍니다."

찰떡이는 6가지 기능을 제공합니다.

  • 사용법 질문 — "포스트잇 색깔 종류 알려줘", "프로젝트 등록하는 법" — 팀웍스 공식 문서를 직접 검색해 답변.
  • 일반 질문 — "오늘 서울 날씨 어때?", "오늘 뉴스 검색해줘" — 시사·날씨·일반 지식까지 웹 검색.
  • 일정 조회 — "오늘 일정 알려줘", "디자인 리뷰 언제야?" — 캘린더 안 들어가도 즉시 조회.
  • 일정 등록 — "내일 오후 3시 주간회의 등록해줘", "13일 11시 반 직원점심 잡아줘" — 한 줄이면 일정 등록 끝. 미리보기 카드 → ✓ 클릭 후 실제 반영.
  • 일정 삭제 — "내일 회의 삭제해줘", "내일 약속 취소해줘" — 대상 일정 식별 → 미리보기 → ✓ 후 삭제.
  • 일정 수정 — "내일 회의 수정해줘" — 수정할 사항을 단계별로 묻고 답한 후 미리보기 → ✓ 후 반영.

다중 턴 대화 — "내일 회의 등록해줘" 같이 시간이 빠진 요청에는 "몇 시에 잡을까요?" 라고 되묻습니다. "5시"처럼 오전인지 오후인지 모호하면 "오전/오후 어느 쪽일까요?" 한 번 더 확인.

찰떡이가 안 하는 일 — 프로젝트·자료실·채팅 같은 다른 영역의 변경 작업은 정중히 거절합니다.

"개떡같이 물어도 찰떡같이 대답하라고 주인님이 지어주셨어요~~" — 그래서 이름이 찰떡이입니다.

VI. Speak & Go (STT) — "불편한 휴대폰 키보드 안녕"

휴대폰 키보드 작아서 답답할 때, 마이크 버튼 한 번이면 음성으로 끝납니다.

  • 마이크 버튼 한 번 — AI 찰떡이 입력창 옆과 팀채팅 입력창 옆에 마이크 아이콘이 있어요. 누르고 한국어로 말하면 텍스트가 입력창에 채워집니다. 검토 후 [전송] 클릭하면 완료.
  • 자동 침묵 감지 — 발화 후 잠시 멈추면 자동으로 변환 시작.
  • 확인 후 전송 — 인식 결과는 입력창에 채워두기만 합니다. 오인식이 있어도 한 번 더 살펴보고 보낼 수 있어 안심.
