# 처음 사용하기

## 회원가입은 어떻게 하나요

1. 메인 화면에서 **회원가입** 링크를 누릅니다.
2. **이름**(최대 50자), **이메일**, **비밀번호**를 입력합니다.
3. **회원가입** 버튼을 누르면 자동으로 로그인되어 팀 목록 화면으로 이동합니다.

이미 가입된 이메일이면 "이미 사용 중인 이메일입니다" 안내가 뜹니다. 다른 이메일을 쓰거나 로그인하세요.

## 로그인은 어떻게 하나요

이메일과 비밀번호를 입력하고 **로그인** 버튼을 누릅니다. 로그인 상태는 두 종류의 토큰으로 유지됩니다:
- **Access Token (15분)** — 매 요청에 자동 첨부
- **Refresh Token (7일)** — Access 만료 시 자동 갱신용

7일 동안 한 번도 활동이 없으면 Refresh Token이 만료되어 다시 로그인해야 합니다.

## 카카오 계정으로 로그인할 수 있나요

네. 로그인·회원가입 화면의 **카카오로 시작하기** 버튼을 누르면 됩니다.

1. **카카오로 시작하기** 버튼을 누르면 카카오 로그인·동의 화면으로 이동합니다.
2. 카카오 로그인 후 **이메일·닉네임·프로필 이미지 제공에 동의**합니다.
3. 동의를 마치면 자동으로 팀웍스 계정이 만들어지거나(처음이면) 기존 계정에 연결되어(같은 이메일이면) 팀 목록 화면으로 들어갑니다.

별도 비밀번호 입력은 없습니다. 카카오로만 가입한 계정은 비밀번호가 없으며, 다음부터도 **카카오로 시작하기**로 로그인하면 됩니다.

> **이메일 동의는 필수입니다.** 카카오 동의 화면에서 이메일 제공에 동의하지 않으면 "카카오 계정 이메일 동의가 필요합니다" 안내가 뜨고 가입이 되지 않습니다. 다시 시도해 이메일에 동의해 주세요. 이미 같은 이메일로 일반 가입한 적이 있으면 카카오 로그인 시 자동으로 그 계정에 연결됩니다(계정이 중복 생성되지 않습니다).

## 처음 들어왔는데 무엇부터 해야 하나요

선택지가 두 가지입니다.

**1) 직접 팀을 만들고 싶다면**
- 팀 목록 화면에서 **+ 팀 만들기** 버튼 → 팀명(최대 100자) · 설명(선택, 최대 500자) · 공개 여부 입력 → 생성
- 자동으로 그 팀의 **팀장(LEADER)**이 됩니다.

**2) 기존 팀에 들어가고 싶다면**
- 팀 목록 화면에서 **팀 탐색** 버튼 → 공개 팀 목록에서 원하는 팀 선택 → **가입 신청**
- 신청은 PENDING 상태로 그 팀 팀장의 "나의 할 일" 목록에 들어갑니다.
- 팀장이 승인하면 팀에 합류, 거절하면 합류 안 됩니다.

## 첫 팀을 만들었습니다. 다음은 무엇을 하나요

팀 메인 화면(`/teams/[팀ID]`)으로 들어가면 **왼쪽에 캘린더, 오른쪽에 채팅**이 동시에 보입니다. 우측 상단에 두 개의 탭이 있습니다 — **팀채팅** / **AI 버틀러**. 채팅 탭 안에는 다시 sub-탭으로 **채팅 / 자료실** 이 있습니다.

첫 시도로 권하는 흐름:

1. **+ 일정 추가**로 다음 회의 시간을 등록 — 팀원 모두에게 즉시 보입니다.
2. **공지 작성**으로 첫 인사말 또는 팀 규칙을 고정 — 채팅 최상단에 항상 표시됩니다.
3. **자료실 sub-탭** → **+ 글쓰기** 로 첫 자료(예: 팀 소개서·회의록)를 글 + 첨부파일로 등록.
4. **AI 버틀러** 탭에서 "팀에 일정 등록하는 법 알려줘" 같은 사용법 질문도 시도해 보세요.
5. 팀원에게 **팀 탐색** 화면에서 가입 신청하라고 알려주세요.

## 여러 팀에 동시에 속할 수 있나요

네. 한 계정으로 여러 팀의 LEADER나 MEMBER가 될 수 있습니다. 팀 목록 화면에서 내가 속한 모든 팀이 한 번에 보이고, 각 팀을 들어가면 그 팀의 일정·채팅·자료실만 보입니다.

## 비밀번호를 잊었어요

현재 버전에서는 비밀번호 재설정 기능이 없습니다. 새 계정으로 회원가입한 뒤 기존 팀에 다시 가입 신청하셔야 합니다. (향후 추가 예정 기능) 같은 이메일을 쓰는 카카오 계정이 있다면 **카카오로 시작하기**로 로그인해 기존 계정에 그대로 접속할 수도 있습니다.

## 무엇이 즉시 다른 사람에게 보이나요

- **일정 추가/수정/삭제** — 팀 캘린더에 즉시 반영
- **채팅 메시지(일반/업무보고)** — 채팅창에 즉시 반영 (3초 자동 폴링)
- **공지사항** — 채팅 상단 고정 영역에 즉시 표시
- **포스트잇** — 해당 날짜 캘린더 셀에 즉시 표시
- **프로젝트 / 프로젝트 일정 / 세부 일정** — 간트차트에 즉시 반영
- **자료실 글·첨부파일** — 자료실 목록에 즉시 반영 (10초 자동 폴링)

## 어디부터 막힐 수 있나요

- **공개 팀 목록에 내 팀이 안 보여요** — 팀 생성 시 "공개" 옵션을 켜야 다른 사용자가 가입 신청할 수 있습니다.
- **일정 종료 시간을 시작 시간보다 앞으로 설정** — 저장 시 오류. 종료 > 시작이어야 합니다.
- **이미 가입 신청한 팀에 또 신청** — 거절됩니다. PENDING 상태인 신청이 있으면 중복 신청 불가.
- **다른 팀원이 만든 일정·자료실 글을 수정** — 버튼이 안 보입니다. 생성자 본인만 수정·삭제할 수 있습니다.
- **자료실에 11MB 파일 업로드** — 거부됩니다. 파일 1개 ≤ 10MB.
- **자료실에 .exe 또는 .svg 업로드** — 거부됩니다. 허용 MIME(jpg/png/gif/webp/pdf/docx/xlsx/pptx/txt/md/zip)만 가능.
- **AI 버틀러에 일정 색깔·설명 수정 요청** — "AI 어시스턴트는 일정 일시·제목 수정만 지원해요" 안내 (조회·등록·삭제·수정 외 필드는 캘린더 화면에서 직접). 일시·제목 변경은 자연어로 가능합니다.
- **AI 버틀러에 "전체 삭제" / "모두 수정" 요청** — 일괄 처리는 차단됩니다. 한 번에 하나의 일정만 처리 가능.
