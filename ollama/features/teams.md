# 팀 생성(등록·만들기) · 가입 · 수정(편집·변경) · 삭제(제거)

> 검색 키워드: 팀 생성, 등록, 만들기, 만드는법, 만드는 방법, 신규팀, 새팀, 추가, 가입, 가입신청, 승인, 거절, 팀 수정, 편집, 변경, 팀 삭제, 제거, 지우기, 팀원, 팀장, LEADER, MEMBER, 강제탈퇴, 팀 검색, 팀 탐색, 비공개팀, 공개팀

## 팀이란 무엇인가요

팀은 TEAM WORKS의 가장 기본적인 협업 단위입니다. 한 팀에 여러 사용자가 속해 일정·채팅·포스트잇·프로젝트·공지·자료실을 공유합니다. 한 팀의 콘텐츠는 그 팀 안에서만 보이고 다른 팀에는 절대 노출되지 않습니다 (팀 단위 격리).

팀에는 다음 정보가 있습니다:
- **팀 이름** — 최대 100자, 필수
- **팀 업무(설명)** — 최대 500자, 선택
- **공개 여부** — `공개`(팀 검색 가능) / `비공개`(개인 일정 관리) 둘 중 하나
- **팀장 (LEADER)** — 팀 1개당 1명. 팀을 만든 사람이 자동으로 팀장이 됨

## 팀 생성(등록·만들기) — 단계별

팀을 **생성**(등록·만들기) 하려면 다음 절차를 따릅니다.

1. 홈 화면(`/`) 의 **"내 팀 목록"** 아래쪽 **`+ 팀 생성`** 버튼을 누릅니다.
2. **"팀 생성"** 모달이 열립니다. 다음 항목을 입력합니다:
   - **팀 이름** (필수) — 100자 이내
   - **팀 업무** (선택) — 500자 이내, 팀의 목적·소개
   - **팀 공개 설정** — `공개`(다른 사용자가 팀 검색 가능) 또는 `비공개`(나만 보이는 개인 일정용) 카드를 클릭해 선택. 기본값은 공개.
3. 하단의 **`팀 생성`** 버튼을 누르면 팀이 만들어지고, 만든 사람이 자동으로 팀장(LEADER) 으로 등록됩니다.
4. 생성 직후 홈의 "내 팀 목록" 카드에 추가되며, 카드 클릭으로 팀 페이지에 들어갈 수 있습니다.

취소하려면 모달 우상단 X 또는 `취소` 버튼을 누르세요.

## 팀 수정(편집·변경) — 단계별 (팀장 전용)

팀을 **수정**(편집·변경) 하려면 본인이 LEADER 인 팀에서 다음 절차를 따릅니다.

1. 홈 화면 "내 팀 목록" 에서 자신이 LEADER 인 팀 카드 우측의 **`수정`** 버튼(연필 아이콘) 을 누릅니다. (팀원에게는 이 버튼이 표시되지 않습니다.)
2. **"팀 수정"** 모달이 열리고 현재 값이 채워져 있습니다.
3. 변경할 항목을 수정합니다:
   - 팀 이름 (필수)
   - 팀 업무
   - 공개/비공개 설정
4. **`저장`** 버튼을 누르면 즉시 반영됩니다.

변경 사항은 팀원 모두에게 즉시 적용됩니다. 팀원은 수정 권한이 없으니 팀장에게 요청해야 합니다.

## 팀 삭제(제거·지우기) — 단계별 (팀장 전용)

팀을 **삭제**(제거·지우기) 하려면 본인이 LEADER 인 팀에서 다음 절차를 따릅니다.

1. 홈 화면 "내 팀 목록" 에서 자신이 LEADER 인 팀 카드 우측의 **`삭제`** 버튼(휴지통 아이콘) 을 누릅니다.
2. 확인 대화상자가 뜹니다. **"정말 삭제하시겠습니까?"** 메시지에 **`확인`** 을 누르면 영구 삭제됩니다.
3. 그 팀의 **모든 데이터** 가 한꺼번에 삭제됩니다 (CASCADE):
   - 팀원 멤버십
   - 일정·포스트잇·프로젝트·프로젝트 일정·세부 일정
   - 일반 채팅·업무보고·공지사항
   - 자료실 글·첨부파일

복구할 수 없습니다. 신중히 결정하세요. 팀원은 삭제 권한이 없습니다.

## 팀 탐색 (다른 팀 가입) — 단계별

1. 홈 화면 **`팀 검색`** 버튼을 누릅니다. (또는 `/teams/explore` 직접 접근)
2. **"팀 탐색"** 화면에 공개로 설정된 모든 팀이 카드로 보입니다 — 팀명, 설명, 팀장 이름, 구성원 수가 표시됩니다.
3. 합류하고 싶은 팀의 **`가입 신청`** 버튼을 누릅니다.
4. 신청은 PENDING 상태로 그 팀 팀장의 **"나의 할 일"** 화면에 들어갑니다.
5. 팀장이 승인하면 자동으로 MEMBER 로 등록되고 그 팀이 내 "내 팀 목록" 에 나타납니다. 거절되면 합류되지 않습니다.

### 가입 신청 중복 정책
- 같은 사용자가 같은 팀에 PENDING 상태인 신청을 동시에 두 개 이상 가질 수 없습니다.
- 이미 그 팀의 구성원이면 신청 자체가 막힙니다 (Conflict 응답).
- 거절(REJECTED) 된 경우에는 다시 신청할 수 있습니다.
- 승인(APPROVED) 되어 이미 멤버가 된 경우 재가입 신청 불가.

## "나의 할 일" — 가입 신청 처리

팀장(LEADER) 을 위한 통합 가입 신청 처리 화면입니다.

1. 홈 화면 또는 팀 카드의 **`가입 승인`** 배지(빨간 N개 표시) 를 누르거나, 메뉴에서 "나의 할 일" 로 이동합니다 (URL: `/me/tasks`).
2. 화면에는 내가 LEADER 인 모든 팀의 PENDING 신청이 한 곳에 모입니다 — 신청자 이름·이메일, 신청 일시, 어느 팀인지 표시.
3. 신청별로 두 가지 액션 버튼:
   - **`승인`** (노란색) — 즉시 MEMBER 로 등록, 그 팀에 합류
   - **`거절`** — 합류 미발생, 신청자는 재신청 가능
4. 처리 즉시 목록에서 사라지고, 헤더의 빨간 배지 카운트가 줄어듭니다.

## 팀원 조회 — 단계별

1. 홈 또는 팀 페이지 헤더에서 팀 카드 우측의 **`팀원 조회`** 아이콘(사람 모양) 을 누릅니다.
2. **"기획팀 팀원 목록"** 모달이 열립니다 — "총 N명" 과 함께 팀원이 카드로 표시됩니다.
3. 각 카드는 다음을 보여줍니다:
   - 이니셜 + 이름 ("나" 표시)
   - 이메일
   - 역할 배지 (`팀장` / `팀원`)
   - 본인 카드: **`이름 수정`** 아이콘 (연필) — 누르면 즉시 표시 이름 편집
   - 팀장이 보는 다른 팀원 카드: **`팀에서 탈퇴`** 아이콘 (문 모양) — 강제 탈퇴
4. 닫으려면 **`닫기`** 또는 X.

## 팀원 강제 탈퇴 — 단계별 (팀장 전용)

1. 위 "팀원 조회" 모달을 엽니다.
2. 내보낼 팀원 카드의 **`팀에서 탈퇴`** 아이콘을 누릅니다.
3. 확인 대화상자에서 `확인` → 그 팀원이 즉시 팀에서 제거됩니다.

제약:
- 팀장 본인은 자기 자신을 강제 탈퇴 대상으로 지정할 수 없습니다 (`팀장은 탈퇴시킬 수 없습니다` 오류).
- 강제 탈퇴된 팀원은 그 팀에 다시 가입 신청해서 합류할 수 있습니다.
- 팀원이 자신의 의지로 팀을 떠나는 기능은 현재 별도로 없습니다 — 팀장에게 요청해야 합니다.

## 자주 묻는 질문

### Q. 한 사람이 여러 팀에 속할 수 있나요?
네. 한 계정으로 여러 팀의 LEADER가 될 수 있고, 어떤 팀에서는 LEADER, 다른 팀에서는 MEMBER일 수도 있습니다. 홈 화면에서 내가 속한 모든 팀이 한 번에 보입니다.

### Q. 팀 정보 변경 후 팀원에게 알림이 가나요?
현재 별도 푸시·이메일 알림은 없습니다. 팀원이 다음에 팀 화면을 열 때 변경된 정보가 자동으로 보입니다.

### Q. 비공개 팀은 어떻게 만들어진 사람을 초대하나요?
현재 MVP 에서 비공개 팀은 가입 신청을 받지 않습니다. 비공개는 본인 전용 일정 관리에 사용됩니다. 향후 초대 기능이 추가될 예정입니다.

### Q. 개인 일정 관리는 어떻게 하나요? / 혼자 쓰고 싶은데 어떻게 해요?
팀웍스의 모든 일정·자료실·채팅은 팀 단위로 격리되므로, 개인 일정 관리를 하려면 **비공개 팀을 하나 만들어 혼자 쓰는 방식**을 사용합니다.

1. 팀 목록 화면에서 **`+ 팀 만들기`** 클릭
2. 팀명 입력 (예: "내 일정", "개인 메모" 등)
3. **공개 여부를 `비공개`로 선택**
4. 만들면 본인이 LEADER 가 되고, 다른 사람은 팀 탐색에서 찾을 수 없으며 가입 신청도 받지 않습니다.

이 비공개 팀에서 일정 등록·수정·삭제, 5색 컬러 라벨, 컬러 포스트잇 메모, AI 비서 찰떡이를 통한 자연어 일정 관리 등 모든 팀 기능을 본인 전용으로 활용할 수 있습니다.

### Q. 내 프로필(이름) 은 어떻게 바꾸나요?
누구나 자신의 표시 이름을 수정할 수 있습니다. 팀원 조회 모달에서 본인 카드의 **`이름 수정`** 아이콘을 누르세요. 양 끝 공백 정리 후 1~50자 범위여야 합니다. 빈 문자열이나 공백만 입력하면 거부됩니다.
