# 포스트잇 (날짜 메모) · 추가(등록·생성) · 수정(편집) · 삭제(제거)

> 검색 키워드: 포스트잇 등록, 추가, 생성, 만들기, 만드는법, 만드는 방법, 만들어, 작성, 메모, 메모추가, 메모등록, 수정, 편집, 변경, 삭제, 제거, 지우기, 색상, 색깔, 5색, 인디고, 앰버, 블루, 에메랄드, 로즈

## 포스트잇이란 무엇인가요

캘린더의 특정 날짜에 붙이는 짧은 메모입니다. 일정과는 별개로 관리됩니다. **시작·종료 시각이 없고 날짜 단위** 로만 관리됩니다. "오늘 우유 사오기", "이번 주 금요일까지 보고서 마감" 같은 가벼운 알림에 적합합니다.

포스트잇에는 다음 정보가 있습니다:
- **날짜** (필수, YYYY-MM-DD)
- **색상** (5가지 — `indigo` / `amber` / `blue` / `emerald` / `rose`)
- **내용** (텍스트)

## 포스트잇 vs 일정 vs 자료실

| 항목 | 일정 (Schedule) | 포스트잇 (Postit) | 자료실 (Board) |
|------|-----------------|-------------------|---------------|
| 시간 | 시작·종료 시각 필수 | 날짜만 | 작성 시각만 |
| 길이 | 분/시간 단위 | 단순 메모 | 글 + 첨부 |
| 용도 | 회의·마감·이벤트 | 가벼운 알림·할일 | 회의록·자료·보고서 |
| 표시 | 캘린더 셀에 막대 | 캘린더 셀에 색상 메모 | 채팅 sub-탭 글 목록 |
| 첨부 | 없음 | 없음 | 1개, ≤10MB |
| 글자 수 | 제목 200자 + 설명 | 텍스트 1개 | 제목 200자 + 본문 20000자 |

회의나 마감처럼 "몇 시부터 몇 시까지" 가 중요한 것은 일정으로, "그 날 챙겨야 할 잡일" 은 포스트잇으로, "팀 전체가 두고두고 보는 자료" 는 자료실 글로 쓰는 게 일반적입니다.

## 포스트잇 추가(등록·생성·만들기) — 단계별

포스트잇을 **등록**(추가·생성·만들기) 하려면 캘린더 상단의 5색 포스트잇 버튼을 사용합니다.

1. 팀 메인 화면 캘린더 상단의 **포스트잇 색상 팔레트** 영역에서 원하는 색상의 **`(색상) 포스트잇 추가`** 버튼을 누릅니다.
   - `indigo 포스트잇 추가` (남색 동그라미)
   - `amber 포스트잇 추가` (주황 동그라미)
   - `blue 포스트잇 추가` (파랑 동그라미)
   - `emerald 포스트잇 추가` (초록 동그라미)
   - `rose 포스트잇 추가` (분홍 동그라미)
2. 캘린더에서 **포스트잇을 붙일 날짜 셀** 을 클릭합니다.
3. 그 날짜 셀에 빈 포스트잇이 생성되고 텍스트 입력 모드가 됩니다.
4. **메모 내용** 을 입력합니다 (Enter 로 줄바꿈 가능).
5. 다른 곳을 클릭하면 자동 저장됩니다 — 별도 저장 버튼 없음.

### 한 날짜에 여러 개
같은 날짜에 여러 개의 포스트잇을 붙일 수 있습니다. 같은 사용자가 여러 색을 섞어 쓰거나, 여러 팀원이 각자 만들어도 됩니다. 캘린더 셀에 색상 메모가 차곡차곡 표시됩니다.

## 포스트잇 수정(편집·변경) — 단계별

포스트잇을 **수정**(편집·변경) 하려면 본인이 만든 포스트잇을 클릭합니다.

**수정 권한**: 만든 사람 본인만. 팀장이라도 다른 팀원의 포스트잇은 수정 불가.

1. 캘린더에서 본인이 만든 포스트잇을 클릭합니다 — 텍스트 영역이 편집 가능 상태로 전환됩니다.
2. 내용을 수정합니다.
3. 다른 곳을 클릭하면 자동 저장.

다른 팀원이 만든 포스트잇은 read-only 로 표시되며 클릭해도 편집 모드로 들어가지 않습니다.

### 색상 변경
현재 MVP 에는 색상 변경 기능이 없습니다. 색상을 바꾸려면 기존 포스트잇을 삭제하고 새 색상으로 다시 만드세요.

### 날짜 이동
자동 이동은 없습니다. 한 번 붙인 날짜에 그대로 남습니다. 날짜를 바꾸려면 삭제 후 새 날짜에 다시 만들어야 합니다.

## 포스트잇 삭제(제거·지우기) — 단계별

포스트잇을 **삭제**(제거·지우기) 하려면 휴지통 아이콘을 사용합니다.

**삭제 권한**: 만든 사람 본인만.

1. 캘린더에서 본인이 만든 포스트잇 위에 마우스를 올리면 우측 상단에 **`포스트잇 삭제`** 아이콘(휴지통) 이 나타납니다.
2. 휴지통을 누르면 즉시 삭제됩니다 (별도 확인 대화 없음 — 가벼운 메모이므로).

다른 팀원이 만든 포스트잇은 삭제 아이콘이 표시되지 않습니다.

## 자주 묻는 질문

### Q. 포스트잇은 누가 만들 수 있나요?
팀 구성원이라면 누구나 (LEADER·MEMBER 모두) 만들 수 있습니다. 일정과 같은 정책입니다.

### Q. 포스트잇 색상은 의미가 있나요?
시스템이 정한 의미는 없습니다. 팀 자체 규약("긴급은 rose, 일반은 amber") 으로 활용하세요.

### Q. 포스트잇이 다른 팀에도 보이나요?
아니요. 그 팀 안에서만 보입니다. 팀 단위 격리는 일정·채팅·자료실과 동일.

### Q. 포스트잇 알림이 오나요?
현재 MVP 에는 푸시 알림 없습니다. 캘린더에 들어와서 시각적으로 확인하는 방식입니다.

### Q. AI 찰떡이로 포스트잇 등록·삭제 가능한가요?
아니요. 찰떡이는 **일정** 조회·등록·수정·삭제만 처리합니다. 포스트잇은 가벼운 메모이므로 캘린더에서 직접 색상 버튼 + 날짜 클릭 한 번으로 충분합니다.

### Q. 한 날짜에 너무 많은 포스트잇이 있으면 어떻게 보이나요?
캘린더 월 뷰에서는 셀 크기가 한정되어 있어 일부만 보이고 나머지는 가려질 수 있습니다. 그 날짜를 클릭해 일 뷰로 전환하면 모든 포스트잇이 보입니다.
