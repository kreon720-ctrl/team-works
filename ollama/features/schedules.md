# 캘린더 · 일정 등록(생성·추가·만들기·잡기) · 수정(편집·변경) · 삭제(제거)

> 검색 키워드: 일정 등록, 추가, 생성, 만들기, 만드는법, 만드는 방법, 잡기, 예약, 일정 수정, 편집, 변경, 일정 삭제, 제거, 지우기, 취소, 캘린더, 월뷰, 주뷰, 일뷰, 회의, 미팅, 약속, 색상, 색깔, 시작시각, 종료시각, KST, AI 찰떡, 자연어 등록, 개인 일정 관리, 혼자 쓰기, 비공개 팀

> **개인 일정 관리 / 혼자 쓰는 캘린더**: 팀웍스 일정은 팀 단위로 격리되므로 혼자만의 캘린더로 쓰려면 **비공개 팀을 하나 만들어 본인 전용으로 사용**하면 됩니다. 팀 만들기에서 공개 여부를 `비공개` 로 선택하면 다른 사람이 찾을 수 없는 본인 전용 공간이 되고, 아래의 모든 일정 등록·수정·삭제·포스트잇·AI 비서 기능을 그대로 활용할 수 있습니다.

## 캘린더는 어디에서 보나요

팀 메인 화면(`/teams/[팀ID]`)에 들어가면 **왼쪽에 캘린더, 오른쪽에 채팅·AI 찰떡이** 가 동시에 표시됩니다 (PC). 모바일에서는 상단 탭으로 `캘린더 / 팀채팅 / AI 찰떡이` 를 전환합니다. 캘린더는 그 팀의 일정만 보여주며, 다른 팀의 일정은 절대 보이지 않습니다.

캘린더 상단 뷰 전환 버튼:
- **`월`** — 한 달 전체. 일정과 포스트잇이 날짜 셀에 표시됩니다.
- **`주`** — 7일치를 시간대별 그리드로 표시.
- **`일`** — 하루를 시간대별로 상세 표시.
- **`프로젝트 관리`** — 프로젝트 간트차트. 활성 시 우측 채팅이 그 프로젝트 전용으로 자동 전환.

## 일정 등록(생성·추가·만들기·잡기) — 단계별

일정을 **등록**(생성·추가·만들기·잡기) 하려면 다음 절차를 따릅니다.

1. 캘린더 상단 우측의 **`+ 일정 등록`** 버튼(노란색) 을 누릅니다. (compact 모드에서는 `등록`)
   - 또는 캘린더에서 빈 날짜 셀을 클릭해도 그 날짜로 일정 등록 모달이 열립니다.
2. **"일정 등록"** 모달에 다음을 입력합니다:
   - **제목** (필수, 최대 200자) — 예: "주간 정기 회의"
   - **색상** (5가지 중 선택) — `indigo`(남색) / `amber`(주황) / `blue`(파랑) / `emerald`(에메랄드) / `rose`(분홍). 클릭으로 선택, 선택된 색은 테두리로 강조.
   - **설명** (선택) — 메모·안건·의제 등
   - **시작 일시** (필수) — 캘린더 아이콘으로 날짜·시간 선택
   - **종료 일시** (필수) — 시작 시각보다 뒤여야 함
3. 하단의 **`생성`** 버튼을 누르면 캘린더에 즉시 반영, 팀원 모두에게 보입니다.
4. 취소하려면 **`취소`** 또는 X.

### 검증 규칙
- 종료 시각이 시작 시각보다 앞이거나 같으면 저장이 막힙니다 (DB CHECK 제약).
- 제목은 200자, 설명은 길이 제한 없음.

## AI 찰떡이로 일정 등록 — 단계별

1. 우측 채팅 영역의 **`AI 찰떡이`** 탭(또는 모바일 탭) 을 누릅니다.
2. 입력창에 자연어로 요청합니다. 예:
   - "내일 오후 3시 주간회의 등록해줘"
   - "다음주 수요일 14시 디자인 리뷰 1시간 잡아줘"
   - "5월 15일 오후 2시 마케팅 검토 미팅"
3. 찰떡이가 인자(제목·시작·종료) 를 추출해 **"다음 내용으로 일정 등록할까요?"** 미리보기 카드를 띄웁니다 — 제목·시작·종료가 한국 시간으로 표시됩니다.
4. 카드 아래 두 버튼:
   - **`승인`** (노란색) → 즉시 등록, 좌측 캘린더 자동 갱신, 채팅에 "완료했어요. 캘린더에 반영됐어요. ✓" 표시.
   - **`취소`** (회색) → "실행을 취소했어요." 안내, 등록 안 됨.

### 정보가 부족하면
- "내일 회의 등록해줘" 처럼 시간이 빠지면 → "몇 시에 잡을까요?" 후속 질문 (다중 턴).
- "1시" 처럼 AM/PM 미명시면 → "오전/오후 어느 쪽일까요? (예: '오후 1시')" 후속 질문.
- "회의 잡아줘" 처럼 날짜·시간 모두 없으면 → "언제로 잡을까요?" 후속 질문.

답변하면 그제서야 미리보기 카드가 뜹니다.

## 일정 수정(편집·변경) — 단계별

일정을 **수정**(편집·변경) 하려면 본인이 만든 일정을 클릭한 뒤 수정 버튼을 누릅니다.

**수정 권한**: 그 일정을 만든 사람 본인만. 팀장이라도 다른 팀원이 만든 일정은 수정 불가.

1. 캘린더에서 해당 일정 막대를 클릭합니다.
2. **일정 상세** 모달이 열려 제목·설명·시작·종료·색상·생성자 정보가 표시됩니다.
3. 본인이 만든 일정이면 우상단(또는 하단) 에 **`수정`** 버튼이 보입니다 (다른 사람 것에는 표시되지 않음). 클릭.
4. 등록 모달과 동일한 폼으로 전환되어 현재 값이 채워져 있습니다. 변경하고 **`저장`** 버튼.
5. 캘린더 즉시 갱신, 팀원 모두에게 변경된 정보 보입니다.

### AI 찰떡이로도 수정 가능합니다
"내일 회의 수정해줘" / "5월 14일 회의 변경" 같이 자연어로 요청하면 찰떡이가 단계별로 안내합니다 — 대상 식별 → 새 일시 묻기 → 새 제목 묻기 → 변경 전/후 비교 confirm 카드 → ✓ 승인 시 PATCH 실행. 변경하지 않을 필드는 **"그대로"** 라고 답하면 됩니다 (예: 시각만 바꾸려면 새 제목 단계에서 "그대로"). 다만 **색깔·설명·메모는 자동 수정 미지원** — 캘린더 화면에서 직접 변경해 주세요.

수정 권한은 일정 생성자 본인만 — 화면이든 찰떡이든 동일.

## 일정 삭제(제거·지우기) — 단계별

일정을 **삭제**(제거·지우기) 하려면 본인이 만든 일정을 클릭한 뒤 삭제 버튼을 누릅니다.

**삭제 권한**: 그 일정을 만든 사람 본인만.

1. 캘린더에서 해당 일정 막대를 클릭 → 상세 모달.
2. 본인이 만든 일정이면 **`삭제`** 버튼(빨간색) 이 보입니다.
3. 확인 대화상자 → `확인` → 영구 삭제, 캘린더에서 사라집니다.

복구할 수 없으니 신중히. 다른 팀원이 참고할 일정은 삭제 전에 채팅으로 공지하는 것을 권장합니다.

### AI 찰떡이로도 삭제 가능합니다
"내일 회의 삭제해줘" / "5월 14일 고객미팅 지워줘" 같이 자연어로 요청하면 찰떡이가 대상 일정을 식별하고 confirm 카드를 띄웁니다 — ✓ 승인 시 즉시 삭제 + 캘린더 갱신. 후보가 여러 건이면 "어떤 일정을 삭제할까요?" 좁히기 안내가 나옵니다. **"전체 삭제" / "모두 삭제"** 같은 일괄 시도는 차단됩니다 — 한 번에 하나만.

삭제 권한은 일정 생성자 본인만 — 화면이든 찰떡이든 동일.

## 일정 조회 — 자연어 검색

별도 검색 UI 는 없지만 **AI 찰떡이로 자연어 조회** 가 가능합니다:
- "오늘 일정 알려줘"
- "다음주 수요일 오전 일정"
- "디자인 리뷰 언제야?"
- "이번 주 회의 일정"
- "5월 1일 오후 일정 뭐 있어?"

찰떡이는 view (day/week/month) + date + keyword 를 추출해 백엔드에서 일정을 조회한 뒤 자연어로 정리해 답합니다. 결과 0 건이면 "(범위) 일정이 없어요." 라고 답합니다.

## 일정 색상은 무슨 뜻인가요

5가지 색상(`indigo`, `blue`, `emerald`, `amber`, `rose`) 중 사용자가 자유롭게 선택합니다. 시스템이 자동 부여하지 않으며 의미도 정해져 있지 않습니다. 팀 내부 규약("회의는 indigo, 마감은 rose" 등) 으로 활용하세요.

## 자주 묻는 질문

### Q. 다른 팀의 일정도 보이나요?
아니요. 절대 안 보입니다. 한 팀의 일정·채팅·포스트잇·자료실은 그 팀 안에서만 노출됩니다. 통합 캘린더 뷰는 현재 미지원.

### Q. 시간대(타임존) 는 어떻게 처리되나요?
모든 일정은 **KST (UTC+9)** 기준으로 입력·표시됩니다. DB 에는 UTC 로 저장되지만 사용자는 의식할 필요 없습니다. AI 찰떡이도 한국 시간 그대로 받습니다.

### Q. 캘린더 날짜를 클릭하면 그 날 채팅도 보이나요?
네. 팀 메인 화면에서 캘린더의 날짜 셀을 클릭하면 우측 채팅 영역이 그 날짜의 메시지로 자동 전환됩니다. 일정과 대화의 맥락을 한 화면에서 빠르게 잇기 위한 디자인입니다.

### Q. 반복 일정(매주 회의 등) 은 만들 수 있나요?
현재 MVP 에는 반복 일정 기능이 없습니다. 매번 개별 일정으로 등록해야 합니다. 향후 추가 예정.

### Q. 과거 시각으로 일정을 등록할 수 있나요?
화면 등록 모달에서는 가능합니다 (회고용 등). 하지만 AI 찰떡이는 과거 시각 등록 요청을 거부합니다 — "이미 지난 시각이에요. 미래 날짜·시각으로 다시 알려주세요." 라고 후속 질문이 나옵니다.
